CPU 究竟是干什么用的

按照 Intel-CPU 的功能和组成的定义 「CPU」 主要有以下功能：

  • 程序控制
  • 操作控制
  • 时间控制
  • 数据加工

如果更加深入一层的话——中央处理器操作原理-维基百科就会知道「CPU」实质上就是负责解析相关的指令与计算相关的数据， 所以选择 「CPU」 主要考虑运算能力！

Intel 与 AMD 发展历史

本段主要梳理两者的发展史，如果想了解历史代表性 CPU请访问【绝世经典】英特尔和AMD的20年（1995~2016）之经典

  • 1981-1985年 蜜月期

    为了赢得 IBM 订单 ，Intel 开放技术，全面授权 AMD 生产「x86系列处理器」，而 AMD 则放弃了自己的竞争产品，成为 Intel 后备供应商。

  • 1985-2001年 拉锯战

    在此阶段以奔腾为代表的Intel和以速龙为代表的AMD在这期间几乎是你超我一点，我超你一点，双方几乎拉不开很明显的差距！

    另外值得注意的一点时我们现在的CPU接口的分流就是在此开始——Intel 改用 Slot1 接口，而 AMD 则在原接口之上推出了 Socket7 接口，并提出了 Super7 架构！

  • 2004-2006年：AMD逆转，最辉煌的时期

    2003年8月，AMD发布了世界上第一款64位民用处理器 Athlon 64。X86拓展指令集正式成为CPU业界标准。 AMD在长达30多年的技术竞争史上首次领先对手，打破了以往技术跟随与模仿者的形象，用64位处理器证明了自己的实力，一场全面的反击战打响了！

    在2005年5月26日，Intel抢先AMD发布了桌面上第一款双核CPU，Pentium D，虽然内部是由两颗Pentium 4共享FSB组成、后来还被证实为「高发热、低性能」，但也是历史上第一款双核。约1周后，AMD拿出了自家的双核Athlon 64 X2。（在这段时间AMD还挑起了著名的“真假双核”的言论）……

  • 2006-2008年：酷睿诞生，Intel全面反击

    从05年开始，Intel就制定了一套“钟摆计划”（Tick-Tock战略），每一个Tick-Tock代表着2年一次的工艺制程进步！

    2006年7月，Intel新一代处理器Core 2横空出世（酷睿2，俗称扣肉），使用65纳米制程工艺，英特尔声称它会有40%的效能增长，同时减少40%的功耗。这一重磅产品的发布，让AMD的Athlon 64 X2瞬间优势全无，性能宝座重新落入Intel手上（直到今天）。

    高端双核市场由Intel重新主导，新的产品定价，让Athlon 64 X2一夜暴降千元以上…… AMD沿用了3年、最成功、最耀眼的FX品牌，开始淡出人们的视野。

    打败了AMD的Athlon 64 X2之后，Intel年底又发布了首款四核CPU，Core 2 Quad，加之 AMD 推出的羿龙存在问题，Intel 重新树立在技术层面的绝对领先位置

  • 2009-2017年：避重就轻，AMD改变路线

    自从酷睿2上市之后，Intel在高端市场一扫千军，无所匹敌，并严格依照Tick-Tock路线实现工艺架构更新，不给对手喘息的机会。如此一来，AMD只能调整战略，以多核心、性价比、整合平台等优势闯出一片天地。

    • 针对Intel的奔腾Dual-Core系列发布新一代的Phenom II和Athlon II，利用多核优势与之周旋
    • 针对Intel在GPU的弱势，2011年，AMD推出APU与「推土机」
    • ……

  • 2017年 AMD 翻身仗 2017年，AMD在沿用了6年的推土机架构后，终于研制出了全新的架构，这也就是大名鼎鼎的Ryzne锐龙处理器！

    2017年2月，Ryzen正式发布，第一代Ryzen的单核心性能比Intel弱一些，其游戏性能不如Intel，加上内存控制器的劣势，很多内存点不亮，而且高频内存频率上不去，所以市场反响并不是很好

    2018年上半旬，Ryzen第二代发布，第二代解决了第一代挑内存的问题，到目前为止，只要不是偏门到型号都不好找的内存，基本上不可能说点不亮，不光内存兼容性问题解决了，第二代的Ryzen单核心性能也有了大幅度的提升，除此之外，内存控制器也允许内存到3200频率了，目前Ryzen同对位的CPU，已经完全可以和intel的Core正面五五开了。

显卡

敲定 CPU 之后关键的一步就是敲定显卡与主板，本段主要是理顺显卡的相关要点！

显卡相比CPU的主要功能在于构图、着色以及光栅化，说的直白一点就是如何渲染出相关的图形以及色彩！

制造显卡的两大巨头是 AMD 和 NVIDIA 公司，两者相比较而言后者要比前者好一点。两家公司整体负责的GPU核心部件，其他显卡厂家负责将显卡 生产出来所以我们才会看到很多品牌的显卡如迪兰……，关于这部分可以阅读为什么芯片型号相同的显卡，不同品牌会有那么大的差价以及主流英伟达显卡品牌推荐与简略分析

NVIDIA 有 GT 和 GTX 系列显卡，GT 是入门级别，GTX 定位更高。后缀有 M 和 TI，M 是移动版本，用于笔记本，性能有缩水；TI 比不带 TI 的性能略强。中间有三位数字，比如TX980，第一位是代数，越高越新，第二位是性能，同代显卡比较的话越高性能越强。第三位一般是0没有意义，也有5，比如 965M 的性能介于 970M 和 960M 之间。

AMD的显卡有HD系列，R系列，RX系列。HD系列停产有一段时间了，不多说。R系列有R9、R7、R5，定位R9>R7>R5。后面有三位数字，意义类似于GTX系列。后缀有X，性能会有加强。最新的是RX系列，后面的三位数字还是跟上面一样。

主板

本文摘录自【小白必看】如何选择合适的电脑主板简化而成，相关的步骤为：

选择 CPU 对应的主板芯片组

英特尔平台芯片组

支持英特尔8代CPU的是300系芯片组的主板，档次从低到高分别为：H310、B360、H370、Z370。

  • H310：入门级，价格最低，适合家用办公。建议搭配奔腾G5400\G5500这类CPU使用。
  • B360：主流级，价格适中，适合后缀不带k的CPU。建议搭配酷睿i3-8100\i5-8400\i5-8500\i5-8600\i7-8700等。
  • H370：中高端，价格稍贵，建议搭配酷睿i7-8700使用。
  • Z370：高端级，价格贵，能超频，支持4000+高频内存，适合超频玩家。建议搭配i3-8350k/i5-8600k/i7-8700k等后缀带K的CPU。（注：英特尔后缀带k的CPU是可以超频的CPU）

AMD平台芯片组

AMD支持锐龙CPU的主板档次从低到高分别为：A320、B350、X370、X470、X399。

  • A320：入门级，价格最低，适合家用办公，不能超频。适合锐龙R3系列CPU
  • B350：主流级，价格适中，适合大部分用户，可以超频。适合锐龙R3、R5、R7系列CPU。
  • X370\X470：高端级，价格较高，有更完善的超频设计和更多的高性能接口，可以组双显卡。两个放到一起说是因为X470是代替X370的新款，但两者区别不大，适合R7系列后缀带X的CPU。
  • X399：高端级，价格昂贵，面向专业用户，仅支持AMD TR4接口的 锐龙 Threadripper（线程撕裂者）CPU。

选择主板版型

要根据机箱的大小选择合适的板型。最常见的是ATX大板和M-ATX小板，大小板性能几乎没有差距，仅扩展性有差别。

  • E-ATX型主板是高性能主板，芯片组多为X299、X399，适用于AMD的线程撕裂者、英特尔的i9-7920X等CPU，价格昂贵，很少见。
  • ATX型主板俗称「大板」，形状类似长方形，这类主板由于更大，所以有更多的插槽和接口，可扩展性较高。比如装双显卡、视频采集卡、PCI网卡等。
  • M-ATX型主板俗称「小板」，形状接近正方形，这类主板该有的接口都有，不过PCI-E插槽数量较少，可扩展性较小。但价格比大板便宜，预算不多的用户可以选择。
  • mini-ITX型主板体型更小，这类主板的接口仅属于刚刚够用，几乎没有扩展性，这种主板仅适用于桌面小机箱，较少见。

选择主板品牌

口诀：高端主板买华硕；中端主板买技嘉、华擎；AMD主板买微星；入门主板买映泰、七彩虹

其详细的特点如下：

  • 华硕：三大板厂之首，主板bios界面做的简单易用。最出名的就是高端的ROG玩家国度系列主板，做工、用料都是顶级，超频玩家首选。华硕的中低端主板价格较贵，竞争力不强，所以买华硕建议买高端的主板。
  • 微星：三大板厂之一，微星AMD芯片主板做的很好，比较出名的是迫击炮系列主板和战斧导弹系列主板，是AMD玩家首选。
  • 技嘉：三大板厂之一，技嘉主板用料扎实，比较出名的是AORUS“雕牌”系列主板，用料奢华，设计感强。
  • 华擎：华擎被称为妖板，各种奇怪的主板比较多。华擎主板最大的特点是供电模块堆料比较夸张，比较出名的是z370 pro4系列和太极系列主板，性价比很高。
  • 映泰：映泰主板价格便宜，面向中低端，做工用料多不错，稳定性很好，适合家用。
  • 七彩虹：七彩虹主板价格便宜，没什么可圈可点的地方，稳定性很好，适合家用。

什么是内存

内存条是一种比固态硬盘更先进的存储技术，它的读写速度比硬盘快上十倍有余，以现在主流的DDR4代内存条为例， 它的读写速度可以达到7000MB/s以上，而传统的机械硬盘最高也就不超过300MB/s，即使是目前 最厉害的M.2 NVME固态硬盘也就3000MB/s的样子

其主要的作用是在「高速运转的CPU」与「低速运转的硬盘」之间搭建一个缓冲区，可以类比京东（机械硬盘）—>本地仓库（内存）—>卖家（CPU）， 所以一旦电脑运转速度慢我们一般都会建议换「固态硬盘/加内存」。

内存核心部件和参数

整个内存的生产流程是：颗粒厂商—>模具/贴牌厂商，相关的品牌厂商如下：

类型 品牌厂商
颗粒厂商 三星、海力士、镁光、南亚（台湾）、梦奇达（德国）、茂德（台湾）、华邦（台湾）
模具/贴牌厂商 金士顿，现代，黑金刚，三星，威刚，芝奇，富豪，金邦，海盗船，宇瞻，金泰克等等

关于内存的核心参数主要是：

  • 内存容量：一般而言越大越好，windows10一般建议配置8G以上
  • 模具类型：DDR3与DDR4，此主要是与主板配合
  • 频率：我们经常看到的2133MHz、2400MHz、3200MHz就是内存条的频率，内存条上的频率可以理解为运输货车的载重量，频率越大，货车一次的载能运载的数据量也就越多
  • 时序：一般用CL表示，用官方的话说就是列寻址所需的时钟周期，时序就是我们这个仓库的物流人员找到货物，并把货物装上车的时间，所以如果是相同频率的内存条，时序CL值是越小越好

电源

以上配件确定后就可以仅剩下电源部分，确定电源部分的步骤如下：

  1. 确定整体功率：电脑中耗电最大的就是显卡和CPU了，占电脑总耗电的90%左右，因此我们只需要把显卡和CPU的功率加起来，再加上100瓦左右的余量给风扇、主板、硬盘等就足够了。
  2. 选购电源
    • 80PLUS认证：80puls认证是一个电源转换率标准，转换率越高也就越省电，虽然不能直接作为判断电源好坏的标准，但有金牌或白金牌认证的都不会太差，可以作为参考
    • 电源方案：主流方案为「LLC半桥 + DC-DC」
    • 看电源的电容：日系电容＞台系电容＞国产电容
    • 看质保时间
  3. 看电源的规格和尺寸
    • 非模组：最便宜，但线材都是固定的，装完电脑后会多出来很多用不到的线头，整理比较麻烦。
    • 半模组：稍贵，必须用到的线材是固定的，而一些扩展性的线头可以拆掉，整理起来很简单。
    • 全模组：最贵，所有的线材都可以拆掉，不仅方便理线，而且简洁美观。
