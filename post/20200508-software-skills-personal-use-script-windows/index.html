Git 环境搭建

基础配置部分

  1. 设置基础信息：用户名与邮箱

    $ git config --global user.name "Your Name"
$ git config --global user.email "email@example.com"

  2. 创建密钥

    $ ssh-keygen -t rsa -C "youremail@example.com"

  3. 成功之后在「我的系统用户」文件夹下出现「.ssh文件夹」，用文本档案打开「id_rsa.pub」，将其填写到对应得应用上

    1. gitlab：https://gitlab.com/
    2. github：https://github.com/settings/keys

Git 命令行

  • 查看当前的远程仓库：git remote -v
  • 查看远程仓库信息：git remote show origin
  • 设置远程仓库：git remote add [short_name] [url]
  • 远程仓库重新命名：git remote rename [old_name] [new_name]
  • 删除某个远程仓库：git remote rm [remote_name]
  • 从远程仓库抓取数据：git fetch [remote-name]
  • 显示所有分支：git branch
  • 创建并切换分支：git checkout -b testing

  • 分支合并：

    git checkout master
git merge hotfix
git branch -d hotfix

  • 查看当前状态：git status

hugo 博客环境搭建

Go 环境配置

Hugo 是基于 Go 语言的，所以首先需要搭建 「Go 开发环境」，其命令行如下：

  1. 下载安装包：https://golang.google.cn/dl/
  2. 配置环境变量：
    • GOPATH：Go的安装目录
    • Path：%GOPATH%\bin

「Nodejs」 环境配置

在本人 Hugo 博客系统中需要用到「hugo-algolia」插件，而此插件是基于 「Nodejs」 ，所以需要搭建「Nodejs」开发环境，其操作步骤如下：

  1. 下载安装包：https://nodejs.org/en/

  2. 修改默认的全局目录

    //全局包目录 
npm config set prefix "D:/node/nodejs/node_global/" 
//全局包缓存目录
npm config set cache "D:/ndoe/nodejs/node_cache/"

  3. 配置全局变量模块存放位置：PATH: D:/node/nodejs/node_global/;

其他配置

  1. 下载并安装 Hugo
  2. 安装「hugo-algolia」：npm install hugo-algolia -g

Ruby 环境搭建

  1. 下载并安装： http://www.ruby-lang.org/en/downloads/

  2. 配置环境变量

    SET RUBY_HOME=D:/ruby
SET PATH=%PATH%;%RUBY_HOME%/bin
SET RUBYOPT=rubygems

Python 环境搭建

  1. 下载并安装：http://www.python.org/download/
  2. 配置环境变量：path=%path%;C:\Python
