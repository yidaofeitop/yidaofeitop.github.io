个人建站—wordpress-基础配置

2020年11月08日
 read
   Categories: 软件技巧

在使用 wordpress 搭建一个官网时，发现网络上的教程一般都比较零散，所以将本次搭建的过程记录以下，可能会分解出多篇，大家可以自行搜索

本篇为开篇，主要内容是介绍基础环境的搭建以及初步的配置。

基础环境搭配

关于 WordPress 的介绍以及优劣势因为每个人都有不同的理解所以不做过多的介绍，我仅仅说明一下我这次的需求以及如何架构的：

  • 表层需求：一个简单的官方网站：美容仪器

  • 内层需求：

    • 官网基础要素：主页、产品页、博客页、联系页
    • 基础环境搭建配置后能够让一个程序小白能够正常的发布文章等（和写 Word 文档一样简单）
    • 官网需要配合邮件开发系统……使用

  • 拆分步骤：

    • 确定域名名称、选择公司官网搭建平台、选择企业邮箱平台
    • 购买域名、企业邮箱平台、搭建公司官网平台基础环境
    • ……

关于如何购买域名大家可以参考个人建站2-为 Github 上的 Hugo 博客绑定个性域名-购买域名、关于企业邮箱平台大家可以参考三大原因让我放弃腾讯阿里云企业邮箱-附国内外企业(域名)邮箱汇总

关于公司官网平台基础环境的搭建主要做如下的事情（以下操作环境均为 Ubuntu 20.04 x64）

  1. 搭建 LAMP 平台：宝塔界面

    wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh

  2. 下载 WordPress 并配置相关文件权限

    wget https://wordpress.org/latest.zip
unzip latest.zip
mv wordpress /www/wwwroot/
chown www:www -R /www/wwwroot/wordpress/wp-content

  3. 修改「wp-config.php 」

    /** MySQL 数据库配置 */
define( 'DB_NAME', 'simplertec' );
define( 'DB_USER', 'simplertec' );
define( 'DB_PASSWORD', 'ai19911017' );
define( 'DB_HOST', 'localhost' );
/** MySQL 数据库表前缀 */
$table_prefix = 'st_';
/* FTP */
define('FS_METHOD','direct');

    PS：如下载的为英文版本，可以从中文网站下载完整安装文件，抽取「wp-content\languages」，上传同样的文件位置并编辑「wp-config.php」，增加 define('WPLANG',"zh_CN");

  4. 基础设置

  5. 安装主题：推荐 「 OceanWP、Sydney」，关于主题的配置大家自行摸索

  6. 「设置-固定链接」配置文章访问路径：推荐使用 「年月/文章名」的固定链接，并同步设置「宝塔界面」中的「伪静态」

  7. 安装「elementor-pro3.0.5」参考链接：Free Download Elementor Pro v3.0.6 All Features Activated
