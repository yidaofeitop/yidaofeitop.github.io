抖音-理论部分-7：矩阵账号

2020年05月10日
 read
   Categories: 短视频

关于什么是矩阵账号网络上有很多的介绍文章，比如说《抖音运营的矩阵玩法详解》（很多文章雷同此文），但是如果你自己做顶层设计的时候那么究竟可能采用怎么样的设计呢？

为什么要做矩阵账号

如果想要理解为什么要做矩阵账号就类似去理解麦当劳、KFC 为什么要开那么多的店面为什么要几个套餐？说白一点就是基于以下几点理由：

  1. 增加曝光量
  2. 降低账号风险
  3. 增加收益
  4. 做好用户人群细分

矩阵账号有几种表现形式

按照陈琦：创始人超级IP之“新媒体矩阵管理与私域流量运营”的说法应该是有如下三类：

  • 新媒体渠道矩阵：横跨图文、音频、视频等内容分发平台，多渠道分发、立体化占位
  • 账号矩阵：多个新媒体子账号共同运营，可辐射不同的人群形成流量闭环
  • 内容矩阵：创建多类型的精品栏目，跨平台联合内容输出

給大家看一下以「果壳」为例：搭建新媒体矩阵细分用户群体的整个矩阵规划：

而本文是针对抖音的矩阵所以就我们将范围缩小至此，看看抖音矩阵有哪些做法

  • 围绕企业、品牌、服务搭建：比如「文弘音乐」、「新东方」
  • 围绕主播、网红搭建：比如说「彭十六」、「慧慧周」
  • 围绕家庭关系搭建：比如说「小金刚＋可爱的金刚嫂＋金刚爸＋金刚妈」
  • 围绕团队搭建：比如典型的「洋葱集团」旗下的账号「代古拉K、办公室小野、毒角show」
  • 围绕某一种成功模式搭建：比如「PPT之光」、「PR剪辑之光」、「Word之光」

矩阵账号如何互动呢？

矩阵账号是用来做什么的？就是账号与账号之间建立连接，通过横向的抖音运营策划，打通粉丝，提升商业价值

那互动自然是如何在账号与账号之间建立链接？那如何建立链接呢？

  • 不同客串合拍视频
  • 评论区互动或@对方
  • 标题区互动或@对方
  • 关注中只有矩阵账号
  • 在个人签名中标注
  • 在直播中互动，推荐小号

这是一般的文章提及到的方式方法，实质上还有一种「话题」，只要你们的流量够大的在某个话题之下基本是你们的内容就可以做到流量的互通

企业的矩阵账号如何做呢？

由于暂时功力有限仅讨论抖音部分，其他平台推荐大家阅读8700字详解，新媒体矩阵搭建不完全指南！

主要分为以下部分：

  1. 账号部分：一般建议分解出「蓝V」+个人账号，至于为什么开蓝V，可以阅读《社群运营-资料库-蓝V》
  2. 内容部分：分解出垂类与泛类
    • 垂类：主要负责通过内容构建公司的整体形象，包含公司软植入、产品知识讲解等比较具象化的东西
    • 泛类：主要以获取比较大的流量为主，只要不损公司形象，在内容上做一定的规范，通过奖励机制带动全员参与，占据某个关键词
  3. 运营指导部分：主要是如下的工作
    • 制定奖励机制并执行
    • 安排培训扶持及优秀案例分析
    • 日常落地跟进：包含团队组件、数据收集、复盘分析、优化指导
