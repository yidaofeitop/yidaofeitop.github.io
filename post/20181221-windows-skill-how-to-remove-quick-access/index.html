Windows 技巧3-保护隐私—清除 「windows 10」 快速访问

2018年12月20日
 read
   Categories: windows 教程

[Windows 10]不知道从哪个版本开始引入了「快速访问」的设计，当你点击任务栏中的「文件资源管理器」 图标时就会默认打开「快速访问」页面，页面中会显示「常用文件夹」和「最近使用的文件」两个项目。这虽然 方便了我们的日常使用，提高了工作效率，但是也会泄露我们的隐私，如下图！

那么如何清除掉「Windows10」开始菜单和主页中的「常用/最近项目」，保护我们的个人隐私呢？

修改「文件管理器」打开「此电脑」而不是「快速访问」

打开任意一个文件资源管理器窗口，在菜单栏中切换到「查看」选项卡，点击右下角的「选项」图标打开「文件夹选项」窗口，如图：

PS：为了进一步保护隐私，你还可以点击「清除按钮」清除文件资源管理器历史记录

彻底删除「快速访问」

修改注册表信息

第一步：打开「注册表管理器」

「Win+R」，输入 regedit 回车，打开「注册表管理器」

第二步：定位位置 使用「编辑-查找」定位到 HKEY_CLASSES_ROOT\CLSID\{679f85cb-0220-4080-b29b-5540cc05aab6}\ShellFolder

第三步：修改「ShellFolder」文件夹的权限
相关的步骤如下：

一般而言仅需要进行「第一步」、「第三步」，但是我在进行操作时，如不进行「第二步」修改权限为「完全控制」

关于修改权限可以参考 更改注册表项所有者权限，获取“完全控制”权限

第四步： 修改「Attributes」值 权限更改为「完全控制」之后，将右侧窗口中的「Attributes」值的数值数据由a0100000修改为a0600000，保存退出注册表管理器

重启资源管理器与效果呈现

重启资源管理器：任务栏右键「任务管理器」，定位到「windows 资源管理器」，右键选择「重新启动」
