2018年11月09日

   Categories: 个人建站

本文主要是记录自了解 Hugo 个人建站后真正开始动手使用 Hugo 的相关体验、todo、优化记录！

在自定义 Hugo 过程中主要参考以下文档，非常感谢这些作者的付出（相关的评论无任何的偏见，仅是阅读的一些体验）

关于参考文档推荐

  1. Hugo 从入门到会用 https://blog.olowolo.com/post/hugo-quick-start/
    基础使用，了解基本运转原理，不涉及任何的开发

  2. @github-heartnn Hugo 基础知识：https://github.com/heartnn/hugo-theme-test
    同上，主要关注点在于通用函数（通用函数建议在看以下第二点推荐之后）

  3. pytool：http://blog.pytool.com/tags/hugo/
    相对来说很容易对于 Hugo 相关开发建立基本的认知，推荐看《Hugo 模板template》、《hugo 模板变量》、《个人博客hugo Taxonomy》，看完以上文章对于模板有基础的 了解就基本可以看懂主题内的 HTML 文档

  4. 官方文档：https://gohugo.io/
    最全，最详细的，可协同 http://www.g-var.com/tags/hugo/ 阅读
