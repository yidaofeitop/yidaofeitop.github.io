打印机-如何设置远程扫描-windows10

2020年01月14日
 read
   Categories: 软件技巧

公司新租用了一台打印机（Aficio MP 3351），租赁公司派遣了一个技术人员过来处理，由于公司的电脑已经基本上全部升级至「 Windows 10」 所以就和他沟通了一下「 Windows 10」是怎样设置远程扫描的！

本文就是我在他们公司的基础之上做的技术汇总，以免有人像我一样用3个小时查找资料之后还是没有任何的头绪！

前置工作

主要的前置工作如下：

  • 电脑端操作：打印机驱动安装、新建一个共享文件夹（建议英文）
  • 打印机端操作：设置打印机IP地址

关于如何设置大家可以参考前文：《打印机-如何设置远程扫描-windows7》

电脑端-建立 FTP 服务器

技术人员推荐给我的软件是：「 Sonic FTP Server V1.06」，使用比较简单，但是我查找一圈愣是没有找到这个软件的官网，所以改用了 比较有名的「 FileZilla Server」，下载网址：https://filezilla-project.org/download.php?type=server

此部分主要操作主要分为2个部分：

  1. 配置 FTP 服务器
  2. 配置 Windows 10 防火墙

配置 FTP 服务器

  1. 打开「 Filezilla Server」，Host默认为 localhost(或 127.0.0.1)，即默认将本机作为 FTP服务器。点击 「 Connect」 连接

    连接成功后(Logged on)，日志面板会出现问题跟警告两行红字，这里的问题跟警告并不会对FTP服务器搭建以及连接FTP服务器有影响，因此不对其处理，如有强迫症的可以参考[1][2]

  2. 设置用户名和密码（用户名为：test 密码为：可空）

  3. 设置共享文件夹（注意把文件的创建……权限打开）

到这里，使用 FileZilla Server 搭建 FTP 服务器就完成了

配置 Windows 10 防火墙

Windows 搜索栏搜索「 防火墙」，选择「 允许引用通过 Windows Defender 防火墙进行通信」，按照下图设置相关程序

按照上图步骤最后定位到 FileZilla Server 的安装文件夹，将 Filazilla server interface.exe 跟 Filezilla server.exe 两个应用程序文件都加入进去！
