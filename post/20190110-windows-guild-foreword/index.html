Windows 教程-前言

2019年01月10日
 read
   Categories: windows 教程

初心

说句实在话，写这个主题主要是基于以下目的：

  1. 在以往的解决亲朋好友电脑问题的时候发现大家的问题都比较雷同，按古人说的「授人以鱼不如授人以渔」，那就需要将相关的知识打散梳理形成一个逐步完善的教程
  2. 有时候查询电脑故障解决方案时常发现查询的1 资料比较陈旧；2 营销为主、干货少，导致查询的资料无用的居多，浪费时间
  3. 信息如果不经过整理分析、输出修正那么就无法内化成自己的知识

目前的主题内容架构如下：

由于个人精力与知识面有限，有相关遗漏错误请大家指正，另外如有比较有意义的技巧大家可以互相讨论交流

其分别对应的文章如下：

主脉络：

  1. Windows 教程1-基础阶段-如何下载官方 ISO
  2. Windows 教程2-基础阶段-如何安装 Windows 系统
  3. Windows 教程3-基础阶段-打造一个高效的文件管理系统
  4. Windows 教程4-基础阶段-好的软件改变你的生活
  5. Windows 教程5-进阶阶段-打造一个安全高效的系统

分支路线：

  • Windows 技巧 1 - 如何创建软连接
  • Windows 技巧 2 - 美化 - 如何去除快捷方式箭头
  • Windows 技巧 3 - 保护隐私—清除 「windows 10」 快速访问
  • Windows 技巧 4 - 美化 - 清除 「windows 10」 图片等 6 个文件夹 / oneDrive
  • Windows 技巧 5 - 保护隐私—关闭并清理 Windows 10 磁贴
  • Windows 技巧 6 - 调优 - 如何关闭 Windows 10 更新
