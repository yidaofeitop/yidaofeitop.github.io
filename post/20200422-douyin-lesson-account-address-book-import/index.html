抖音-账号建设部分-1：通讯录导入

2020年04月22日
 read
   Categories: 互联网运营

如果对于抖音的推荐机制有一定了解的人应该都知道有部分的流量是来自于通讯录，加之抖音也会通过算法判断你是否是「群控」，所以在开始注册抖音之前一般来说都必须导入通讯录！

通讯录导入一般来说有如下几种方式：

  1. VCF 导入
  2. 借助工具 APP 导入

以下就这两种具体的方式做展开！

工具 APP 导入

市场上关于此类的通讯录同步软件很多，如QQ通讯录、和通讯录……，本次使用的就是「和通讯录」导入

  1. 安装「和通讯录」 APP，如矩阵苹果手机可采用「爱思助手」直接安装
  2. 手机号注册账号，采用动态码注册，以便后续管理

  3. 点击「导入导出」—>「下载导入模板」

  4. 针对 「名称/手机号」 输入数据，如下图所示（注意右图情况）

  5. 点击通讯录即可看到已经导入的数据

  6. 手机端登录「和通讯录」，点击「我」—>「下载本地」即可

PS：通过以上的错误数据的实践大家可能会发现此方法仅能够针对数据非常有序的情况，所以针对此写了一个小代码，也就是后面的 VCF 导入方式

VCF 导入方式

大家可以访问：https://github.com/yidaofeitop/simpleCode/tree/master/js/concatsFormToVCF

使用时注意依赖库是否同步下载或者有联网，如无的话请用「文本档案」打开，调整此段：

<!--导入 JQuery 库 导入 js-xls插件，如无网络则采用下面方式 
<script lang="javascript" src="https://cdn.bootcss.com/jquery/3.4.1/jquery.min.js"></script>
<script lang="javascript" src="https://cdn.bootcss.com/xlsx/0.15.1/xlsx.full.min.js"></script>-->

<script type="text/javascript" src="dist/jquery-3.4.1.min.js"></script>
<script type="text/javascript" src="dist/xlsx.full.min.js"></script>

使用教程如下：

  1. 按照表格数据类型选择不同的按钮上传 Excel 文件
  2. 登录 icloud.com ，选择通讯录—>VCF 导入即可
  3. 手机端登录 icloud，系统会自动同步
