Windows 技巧1-如何创建软连接

2018年12月16日
 read
   Categories: windows 教程

软连接

此需求来自 MaoXian Web Clipper

此插件只能够访问 Chrome/Firfox 浏览器的下载文件目录，但是由于本人需要将相关的文件存储到对应的需要同步的网盘文件夹，所以采用软连接：

命令行如下：mklink /D D:\download\mx-wc E:\02resource\webCrawling

扩展部分

关于 windows 软链接、硬链接、快捷方式的差别如下：

软连接：

  1. 执行命令 mklink /D link_name target_name
  2. 在系统中不占用空间，在文件系统中不是一个单独的文件
  3. 如果源文件被删除了，链接就没用了；移除源文件不会影响符号链接；移除链接文件也不会影响源文件

硬连接：

  1. 执行命令 mklink /H link_name target_name
  2. 在系统中占用的空间与源文件相同，但在系统中引用的是相同的对象（不是拷贝）
  3. 移除源文件不会影响硬链接；移除硬链接不会影响源文件；如果源文件被删除，它的内容依然通过硬链接存在 硬链接文件的任何更改都会影响到源文件

快捷方式：

  1. 创建方式：在选择的源文件上鼠标右键，通过下拉菜单创建
  2. 快捷方式在系统中跟源文件是完全分离的
  3. 如果源文件删除，链接就没用了；移除源文件不会移除快捷方式；移除快捷方式不会影响到源文件
