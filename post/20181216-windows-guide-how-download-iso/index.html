Windows 教程1-基础阶段-如何下载官方 ISO

2018年12月16日
 read
   Categories: 软件技巧

大家重装系统会去哪里下载镜像呢？百度搜索是不可能的，一辈子都不可能！（具体见《百度竞价排名思考 - 血友病》）

后续查询相关的资料整理如下2种比较好的方式给大家：

  1. MSDN，我告诉你
  2. 微软官方——手机页面

前者非常简单选用相应的版本直接下载即可
后者则是利用浏览器模拟手机端的方式获取下载地址，但是截至2018年12月16日为止 仅适用「Windows 10」 具体操作请见下文。

Windows 10 下载中心：https://www.microsoft.com/zh-cn/software-download/windows10

《如何下载win10的iso?》作者非常细心地注意到 这台电脑 推测是以 user-agent 判断

所以我们采用浏览器调试工具模拟手机端，具体步骤如下：

第一步：chrome 浏览器：页面右键->检查->手机模式
其他浏览器请自行搜索

第二步：选用相关的系统版本即可
