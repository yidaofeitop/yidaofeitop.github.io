2018年11月15日 指纹锁安装总结

2018年11月15日
 read
   Categories: 指纹锁

2018年11月13日 接了一个朋友的安装单，此次安装过程中出现了以下3个最大的问题：

  1. 锁体与门原有锁体不配—》第一天 GAME OVER
  2. 配件包缺弹簧配件—》第二天完成基本安装，第三天追加弹簧配件
  3. 天地钩很难稳定的挂上去

以上安装问题在2018年11月15日 9:05基本解决，仅剩由于天地钩难挂开的一个孔等待快递到达！

2018年11月13日安装流水账

接到此单的时间是2018年11月11日，预定安装为2018年11月12日，由于客户有事所以时间调整为2018年11月13日。

安装流水账如下：

  1. 12:30 出发上车，15:15 到达客户地址
  2. 拆除旧锁，出现后续才想明白的第一个问题点：朋友配置的锁体为 6054 30*388的大霸王锁体
  3. 挂天地钩—》由于 侧边条宽30锁体很容易就掉入门内，采用先固定锁体，在侧边开孔尝试挂天地钩
  4. 锁体固定孔距不对应，采用重新开孔的方式用拉铆钉固定—》 又一个错误：应采用麻花钻4.2开孔，再用5.0丝攻
  5. 用大方钢穿过锁体测试，发现出现天地钩很容易出现掉的情况—》天地钩在开锁与关锁的时会跳动掉落
  6. 将锁体天地钩上钩往下打，下钩往上打，初步测试通过
  7. 开始正式安装锁，再次测试，上天地钩脱落
  8. 再次将上天地钩向下打—》锁体出现卡顿:😢: —》最后发现是敲击过程中将天地钩贴在锁体面板卡顿了
  9. 再次装回旧锁，再次出现锁体无法弹出—》排查发现是侧锁倒装了无法弹出，解决
  10. 试用几次后 上天地钩 再次脱落
  11. 发现是由于 锁体上天地钩不平，门上天地钩在下行的时候门天地钩下行速度快于锁体上天地钩速度，由于 锁体上天地钩为向下倾斜，门天地钩向左边滑动脱落
  12. 解决回家，到家时间为11:00左右

2018年11月14日安装流水账

13日晚上已确认公司有 6054锁体，所以安排在上午再次去安装

安装流水账如下：

  1. 8:20 公司跟单上班，拿到 40*388的6054的锁体（后续才确认为 7554大霸王锁体，安装过程中证明是错的很好）
  2. 11点左右到达客户地址，顺利的拆除旧锁
  3. 上天地钩，发现锁体轴线往门内移动，开主方钢孔位
  4. 反复测试，天地钩无任何脱落问题，斜对角固定锁体
  5. 由于锁体固定钻头很容易碰撞锁体，所以采用用后封板定位相关的孔位，拆下锁体，逐步的开孔
  6. 固定锁体，安装前后面板，测试OK，完成交付:😄:

但是在回程过程中回忆起 配件包无弹簧，且没有用其他方式处理后把手，查看以前的资料发现会容易出现 后把手与方钢存在空隙->方钢滑动->前面板与方钢无任何接触->任何方式无法开门情况，所以 所以出现2018年11月15日再次到达在后把手加入弹簧！

安装总结

  1. 出发前必须要到所有的具体信息：原有门的情况、客户住址、联系方式、锁配件齐全（关键注意门厚与方钢、弹簧关系）、异常方案
  2. 安装一旦出现锁体侧边条不对应，如无丝攻套件，请判断天地钩能否正常处理，如不行请直接拒绝，否则基本很容易亏
  3. 锁体天地钩尽量不要做任何的变形处理，如处理了请在固定锁体之前确认锁体运转正常—》参考 2018年11月13日 第8点
  4. 安装过程中一定要算一下安装完成后方钢不滑动，或者有滑动但是能够确保前面板与方钢一定有接触
  5. 安装完成后检查 关键配件（方钢+弹簧）是否使用、使用是否恰当（注意是两个步骤），并反复检查整体运转情况

进一步关于如何选用指纹锁

  1. 功能选项确认
    指纹锁功能选用：指纹、密码、刷卡、钥匙、防猫眼、APP联网（ZIGBEE 或蓝牙或 WIFI）、蓝牙开锁、微信开锁、临时密码、周期密码、智能家居联动……
    我将功能划分为以下几类，逐一的升级：
    第一层级：必需 家用：指纹、密码、卡、钥匙，酒店：密码、卡、钥匙 第二层级：可加 临时密码、周期密码（非联网方式实现，不需要借助网络）
    第三层级：APP联网（ZIGBEE 或蓝牙或 WIFI）、蓝牙开锁、微信开锁、防猫眼
    第四层级：智能家居联动
    以上层级大家可按照自己的需求逐步的增加，第二层成本增加极少，第三层与第四层成本增加极多，大家按照自己的需求配置
  2. 功能参数选项
    关于指纹头选用请考虑使用环境，如农村建议使用光学指纹头，关于滑盖与直板建议使用直板，滑盖增加隐患点，成本也有增加
  3. 选用锁体
    确定以上参数之后可以用价格、功能参数去淘宝或京东、线下门店去选择自己喜欢的，但是在这里我必须进一步说明大家尽量选用 6068 正装锁体且锁芯为真锁芯 的智能锁
    关于选用6068锁体是因为大家 原有的锁体基本是6068锁体，安装改动少，选用真锁芯是因为只要天地钩 安装无问题（不脱落）就算出现方钢滑动等均可开门，稳定性更加高！
  4. 关于全自动与半自动 从事智能锁行业不久也就2年多，但是经历过智能锁行业从配件全部自行生产->所有部件可外包专业定制->人脸识别/全自动火爆…… 但是对于全自动锁一直持以否定太多，因为全自动问题太多了，如 适配天地钩问题、开锁关键部件稳健性低、紧急供电要求高、噪音大、安全性低…… 半自动的问题呢：锁体适配（售前可解决）、安装要求远小于全自动……，可以说只要安装师傅负责基本上会用的舒舒服服的！

综上建议大家选用具有以下特色的指纹锁：

  1. 四合一，最多加临时密码功能—》绝对能够解决大家99.99%的生活需求，炫酷的功能如人脸识别、掌静脉都是锦上添花，使用频率并不高
  2. 正装6068锁体 真锁芯
  3. 半自动锁（个人看法，大家硬要全自动请选择靠谱的店比较好）
