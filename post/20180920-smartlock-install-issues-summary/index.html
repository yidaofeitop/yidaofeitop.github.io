指纹锁安装常见问题汇总

2018年09月20日
 read
   Categories: 指纹锁

本文主要是针对指纹锁安装过程的一些常见问题的汇总与汇总，应该已经包含了所有的！

后续如是简单的问题则继续加入进来，如涉及环节比较多的则直接开单篇文章！

问题1：上抬把手锁舌缩进去，下压锁舌无反应

类似异常情况有：

  1. 上抬把手锁舌无动作，下压把手锁舌缩回；
  2. 上抬下压把手均无动作

主要原因是：

  1. 方钢套未旋转到位，装锁时请将方钢套上小点或三角形标志与把手同一个方向。
  2. 如电子锁体检查锁体是否与门开门方向一致。

问题2：安装完成后密码或指纹验证成功开不了门

  1. 验证成功后听下电机是否有转动声音，如没有电机转动声音，如使用电子锁体则检查锁体连接线与前面插座是否插牢或插歪。
  2. 检查内外面板的执手方轴方向是否有按正确的方式安装到位。
  3. 检查门外锁体插方轴小头的插口位置是否正确。
  4. 检查门外的执手方轴是否过短或执手方轴推簧没放，导致方轴脱出锁体或前面板方钢孔。

问题3：安装完成后不需要密码指纹等可以直接开锁

  1. 如为机械锁体则先检查方刚套未旋转到位，不到位的正常现象为上抬把手锁体缩回下压锁体无任何动作。
  2. 检查系统是否打开常开模式，如有则关闭。
  3. 如使用机械锁体则用指纹等电子方式开启一次，使面板电机离合分离，如使用电子锁体则可能锁体与面板连接线未连接好导致电机未通电（电子锁体出厂电机离合处于合同状态）
  4. 查内外面板的方轴是否太长以及前后连接螺丝过紧导致门内外的转轴卡死

问题4：安装完成后出现把手回弹力减弱/卡顿

具体的异常情况如下：

  1. 把手下压或上抬之后回位过慢或直接不回位；
  2. 把手下压感觉明显的卡顿

情况A:

  1. 前后连接螺丝规定过紧—逐步松固定螺丝，逐步测试是否可以。

  2. 方钢过长导致有抵触力—安装注意后孔留一定活动空间即可。 情况B:

  3. 如有天地钩则可能是天地钩生锈、孔位未对准等引发——天地钩对准孔位、除锈剂润滑油除锈润滑

  4. 如无天地钩则可能是锁舌与扣板孔位出现偏差或过紧导致——调整扣板位置或更换扣板

  5. 锁体内部### 问题导致——更换锁体

问题5：不读指纹或指纹识别不灵敏

  1. 用干净纸巾擦拭指纹头后在按指纹。
  2. 指纹头排线振动过大可能导致的松动。
  3. 指纹头坏。

问题6：使用过一段时间后面板忽然不供电

  1. 检查电池是否有电。
  2. 检查是否有不同品牌或新旧电池混用导致漏液造成电池盒与后线路板被腐蚀。
  3. 检查锁是否有受潮导致内外面板连接线插座接触不良。
  4. 检查面板连接线是否有松动，导致接触不良。

问题7：锁一直重复回复出厂设置

  1. 电池漏液导致后线路板恢复出厂设置连接短路。
  2. 后线路板受潮或进水导致连接线插座短路。
  3. 恢复出厂设置按键的连接角受潮或漏液导致短路。
  4. 恢复出厂设置按键坏。

问题8：电池耗电快

  1. 检查锁是否有受潮。
  2. 检查是否有不同品牌或新旧电池混用导致漏液造成电池盒与后线路板被腐蚀。
  3. 检查安装过程中是否有螺丝将连接线割破并接触

问题9：验证成功无法开门

  1. 验证成功后，听下电机是否有转动声音，如没有并锁体为机械锁体则可能是安装时过紧大致压迫到电机从而产生电机驱动芯片故障，如没有并锁体为电子锁体则可能是电机连接线长时间被门和面板夹住导致连接线段或线路板电机驱动芯片故障。
  2. 验证成功后电机有转动声音执手往下压是空转，如空转有可能锁体离合卡住或有异物掉进锁体卡住离合器，或执手方轴脱出锁体，执手上抬下压及振动门或面板后再开门反复操作可以先开门。
  3. 验证成功后执手往下压，压不动，说明锁体卡住或锁舌被门卡住，可以控制力道执手下压的同时推拉门来使门与锁舌松动后开门。

问题10：斜舌卡住

  1. 斜舌是否与门磨擦变形导致，（把斜舌拉杆.打上润滑油后试试）
  2. 三叉斜舌中间的推片与门有摩擦导致卡。
  3. 斜舌拉片与门磨擦导致卡。

问题11：主锁舌卡

  1. 锁体安装时导致天地拉杆变形后卡。
  2. 主锁舌伸出状态下与门正面碰撞导致主锁舌变形造成卡。
  3. 木门的木屑或防盗门有异物掉进锁体里面造成天地拉杆卡

问题12：如何判定门是否有天地钩，以及天钩固定方式

一般而言木门由于内部空间### 问题无天地钩，不锈钢以及铁门等情况则相对比较复杂， 判定天地钩的主要方法是：

  1. 第一步将锁体抬出，即关门状态
  2. 第二步用手摸门顶部、侧边，是否有铁杆，如有则安装必须安装天地钩

备注：关于天地钩固定方式一般而言就两种：a 挂；b螺丝固定 如果观察到侧边条除去固定孔以外在上下均有孔位并且有螺丝固定，则可以判定使用的锁体为宝德安-13型锁体

问题15：两道门如何选择

两道门主要需要处理2个问题：

  1. 静止状态下，前后把手是否会重叠
  2. 使用状态下，前后把手是否会碰撞

第一个问题需要测量两道门之间的间距以及后面不换锁的把手高度，前者减去后者就是可以安装锁的后面板高度

第二个问题就是如果前后把手还有间隙，则无需进一步处理，如无则需要安装的指纹锁做到前后不联动
