Windows 教程2-如何安装 Windows 系统

2018年12月17日
 read
   Categories: windows 教程

Windows 安装大致可以分为以下几种情况：

  1. 终极效果
    • 单系统安装
    • 双系统安装（Linux 与 Windows 并存/Mac 与 windows）
  2. 安装方式
    • 外部介质安装
    • 硬盘安装

由于双系统安装（非虚拟机）一般都是专业人员需求，所以本文不做过多的论述，仅推荐 Google 搜索的教程文章：

[1]Mac上安装双系统（传统BootCamp傻瓜式）图文教程
[2]Windows + Ubuntu 16.04 —> 双系统

本文仅讨论 Windows 单系统 安装

我下载的系统合适吗？

为什么会提出此项，主要 a：我们要将硬件效益最大化 b：避免出现辛苦下载的镜像「不适用硬件」

确认内存大小/CPU指令集/BIOS 引导程序种类

首先我们要了解以下知识：

  1. 32位系统最大识别 4G 内存，64位系统可识别 4GB以上内存
    具体原因可访问：32位系统只能寻址4G空间，64位则是128G，这些是怎么算出来的？ - 北极的回答
  2. 64位系统要求 CPU 必须含有「AMD 64、EMT-64、IA-64」其中一项指令集
    具体内容可访问：x86-64-维基百科

  3. 「UEFI+GPT分区」/「Legacy+MBR分区」差异

    特性 UEFI Legacy
    分区方式 GPT MBR
    分区数 128主分区 4主+1逻/5主
    磁盘最大容量 >2TB <2TB
    不支持系统 Win7 32位及之前版本

    具体内容可访问：

    • BIOS, UEFI, MBR, Legacy, GPT等概念整理
    • BIOS+MBR与UEFI+GPT启动模式支持安装的windows系统版本：系统说明「EFI/BIOS」与操作系统的问题
    • UEFI 固件-微软官方：关于 UEFI 位数与系统版本

    注意
    在 UEFI 模式下，Windows 版本必须与电脑体系结构相匹配。64 位 UEFI 电脑仅可以启动 64 位版本的 Windows。32 位电脑仅可以启动 32 位版本的 Windows。在某些情况下，在传统 BIOS 模式下，你可能 能够在 64 位电脑上运行 32 位 Windows，前提是假设制造商支持电脑上的 32 位传统 BIOS 模式。

所以我们必须进行以下行动确认系统硬件的相关情况：

  1. 确认当前系统的内存大小
  2. 确认当前系统 CPU是否有「AMD 64、EMT-64、IA-64」其中一项指令集
  3. 确认当前系统 BIOS 启动方式

关于系统内存确认可以最为通用的方式如下：

  1. 「CTRL+ALT+DELETE」（即所谓的热启动）—>选择「任务管理器」
  2. 切换最上方第二栏的标签，选择「性能」，即可查看当前的内存是多少

关于系统 CPU指令集采用最为标准的测试方式：「CPU-Z」测试

关于查看当前 BIOS 启动方式有以下两种方式：

  1. Windows 8及以上操作系统
    按「Win+R」打开运行，输入msinfo32，回车查看系统信息。
    在BIOS模式中如果显示“传统”，表示系统启动方式为Legacy BIOS;如果为UEFI，则显示UEFI。

  2. 通用方式 按「Win+R」打开运行，输入cmd，右键点击「以管理员方式运行」，输入bcdedit /enum {current}

选用合适的系统

通过上文你可以知道自己当前电脑的内存大小、CPU指令集以及 BIOS引导程序是哪种！

相关的选择如下：

序号 CPU 是否支持64位 内存大小 BIOS 启动方式 建议
1 否 - UEFI 建议将 BIOS 改用「Legacy」启动方式，可安装32操作系统更多
2 否 - Legacy 选用32位操作系统即可
3 是 - UEFI 32位 选择Win7 32 位之后32位操作系统
4 是 - UEFI 64位 选择Win7 32 位之后64位操作系统
5 是 >4 GB Legacy 选择64位操作系统
6 是 <4 GB Legacy 选择32位操作系统

注意点：

  1. 将 BIOS 启动方式修改安装系统之前，请注意将 数据做好备份，因为需要对应的修改分区方式
  2. 关于「UEFI」位数，行业内默认的规则是：「平板电脑一律只支持32位的 「UEFI」 启动，笔记本电脑、台式机电脑一律只支持64位的 「UEFI」 启动」

好了，现在确定你要安装的系统是哪个！那么如何下载官方纯净的系统呢？请访问《Windows 教程1-如何下载官方 ISO》

正式安装

外部介质安装

外部介质安装一般有 DVD 刻录盘、U 盘安装方式，但是由于以下两个原因现在的外部介质安装主要是指「U 盘安装」

  1. 现在的笔记本由于便携性等要求已将 DVD 驱动器删减，只能通过外设方式读取 DVD 盘
  2. 光盘保存麻烦而重装操作系统的频率低

准备工作

相关的准备工作如下：

  1. 一个 8G U盘（如是安装 「Windows 7」 4GB也足以）

  2. 选择「微PE」版本并下载（此处注意必须选用 与 「UEFI」 位数一致的版本）

    摘录官方规格说明：http://www.wepe.com.cn/learnmore.html

    EFI启动时 （32位）只可安装到32位系统开机启动项 （64位）只可安装到64位系统开机启动项

  3. 准备相关的网卡驱动

    • Windows 8与Windows 10本身配置有万能网卡可忽略
    • 其他版本建议直接下载驱动精灵万能网卡版
    • 生产商官方网站

  4. 准备相关的激活工具：不推荐，大家自行搜索

安装工作

安装工作大致分为以下部分：

  1. 将系统变为 U 盘引导启动
  2. 正式安装工作
系统 U 盘启动

第一步：将「微PE」安装到 U 盘

第二步：将前面下载 「Windows 系统镜像」复制到 U 盘

第三步：修改启动引导介质

  1. U 盘插入电脑，电脑重启，在开机几秒内，按下快捷启动热键「F1」（不同的机型不同，请看本文最后的快捷键列表）

  2. 通过左右方向键定位到「Startup」—>「Boot」，然后将带有 USB 字样或者是 U 盘的品牌名（Sandisk、Toshiba 之类）通过「+，-」键移动变为第一项

  3. 按下「F10」保存修改，重启电脑，一般重启后会出现如下界面

第四步：选择 「Win10PEx64/Win8EX64」会进入对应的 PE 界面

正式安装工作

按照是否使用「原生 ISO镜像」可以分为「原生安装」与「安装器安装」，由于原生安装与硬盘安装雷同，所以本处主要介绍「安装器安装」

具体步骤如下（本节内容摘抄自安装器方法安装原版Windows-wePe U盘使用教程,如有侵权请联系本人处理）：

第一步：格式化 C 盘并分区 此处使用的软件可选「分区工具 DiskGenius」或「分区助手（无损）」，推荐使用后者，另外此处可以在本地系统提前处理好！

相关的详细操作请查看 傲梅分区助手官方无损分区教程

关于分区可参考如下「256G 固态硬盘分区方案」：

盘名 大小 原因 追加
C 盘 50 G 1. 系统盘分离可防御如系统崩溃重装导致资料丢失风险 2. Windows系统安装完成一般为35 G 最大60G
D 盘 80 G 1. 公共资料盘，主要用于存放程序安装文件、公共资料 2. 系统重装部分软件无需重新配置，节省时间
E 盘 80 G 1. 私人资料盘，主要用于存放个人资料——利用 windows 自身的 「BitLocker」加密，需密码访问
空白 27.22 G 1. 用于后期备用

第二步：PE 内将要安装的系统 ISO 镜像挂载为虚拟光驱

从U盘启动进入PE内，打开U盘内的存放系统镜像的“ISO”目录，找到要安装的镜像

对着ISO镜像点右键，选择挂载，或者直接双击，该ISO就会被挂载为一个新的虚拟的光盘驱动器，和插入安装光盘效果类似。它会显示在“我的电脑”里，并且有一个盘符。

第三步：Windows 安装器输入选项安装

执行桌面上的Windows安装器，其顶部有两个选项卡，右边的支持WIN10、8.1、7 的安装，左边的支持 XP 的安装。请注意区别对待。

在第一个选项中，请选择系统包 install.wim 的位置（在刚才挂载出来的虚拟光驱里的 「source」 文件夹里），比如 H:\sources\install.wim 等等。

注：如果是安装XP的话，选择左侧的选项卡，第一个选项选择挂载出来的虚拟光驱根目录或 I386。

其余的两项填写要安装到的目标系统盘，正常情况下选择 C: 即可

点击「优化调整」按钮进行相关的系统优化，如图：

最后点击「开始安装」，这时会弹出「都准备好了吗？」窗口，如图：

选择「使用Bootsect.exe更新引导」，支持「 BIOS 和 UEFI」，默认使用「ALL选择项」 如果想要与电脑上现在已有的系统组建成双系统或多系统，则需要勾选「查找并添加已经安装在此电脑的 Windows 版本」

第四步：等待系统安装完成，做最后的系统配置

第五步：进入桌面，装驱动装软件

进入到桌面后，使用提前备好需要安装的驱动先把网卡驱动装了，再联网装配相关驱动，如想要简单快速可以直接使用驱动精灵

硬盘安装

硬盘安装主要是针对在需要重装系统时，发现电脑没有光驱，身边也没有优盘或移动硬盘的尴尬场面！

「NT6 HDD Installer」 安装

「NT6 HDD Installer 」是一款免费实用的 Windows 硬盘重装工具，可以支持在 WinPE、XP 到 Windows 10 等几乎全部 的系统上重新安装 Win10、Win8/7、Server 2012/2008、Vista (NT6.x内核版本的)系统，支持32与64位，你可以格式化 C 盘安装成最纯净的系统，也可以装成多系统，加上硬盘本身读取速度比光盘等介质更快，所以在原系统没有彻底挂掉前用其重装可以说是最佳选择……

本方法不适用于「Windows 10」，具体原因请访问windows10系统无法用nt6 hdd installer了

准备工作

相关的准备工作如下：

  1. 「NT6 HDD Installer」 安装器

    • 其他版本建议直接下载驱动精灵万能网卡版
    • 生产商官方网站
  3. 准备相关的激活工具：不推荐，大家自行搜索

正式安装工作

相比 U 盘安装，硬盘安装相对而言比较简单一点——省略「微PE」安装到优盘步骤！

第一步：将「ISO镜像」解压到除 C盘外 的任意磁盘的根目录中（尽量是一个空白的盘，以免由于部分文件重名覆盖情况导致文件不全）

第二步：右键管理员身份运行「nt6 hdd installer」，安装模式2；

根据作者的说明：
1. 模式1为 「grub4dos方式」，不排除小部分机器不兼容「grub4dos方式」；
2. 模式2为微软自带的引导方式，进系统安装界面的可靠性更高。

当原系统为 2000/XP/2003 时模式1添加启动项的安全性高于模式2，此时推荐选择安装模式1。
当源系统为vista/2008/win7/2008r2时进入系统安装界面可靠性高于模式1，所以推荐使用模式2

第三步：系统配置，进入桌面，装驱动装软件
操作过程中注意定位 C盘所在位置，不要乱动即可！

「WinNTSetup」 安装

准备工作与「NT6 HDD Installer」 安装雷同，主要是下载的文件不一样：Winntsetup Download

安装步骤与「外部介质安装-安装器安装」一致！
