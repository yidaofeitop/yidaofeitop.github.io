个人建站-Hugo 集成 Algolia 搜索

2018年11月19日
 read
   Categories: 个人建站

Algolia 是为你的 APP 或者网站添加搜索的最佳方式。 开发人员可以使用 API 上传并同步希望搜索的数据， 然后可以进行相关的配置，比如产品转化率等等。可以使用 InstantSearch 等前端框架进行自定义搜索，为 用户创造最佳的搜索体验

本文主要参考Hugo 集成 Algolia 搜索

前半部分关于 Algolia 官网操作细节请大家直接访问上面的参考文章，本文主要是针对性补充相关的元素：

  1. 增加关于配合 JS：autocomplete.js 相关说明
  2. “ ADMIN API Key”等可动态配置

Algolia 官网操作以及本地插件

前往Algolia 官网 按照Hugo 集成 Algolia 搜索 注册账号，建立 Indices 索引！

Algolia 提供 API文档进行相关的开发，为简单起见本次使用 hugo-algolia 的插件来完成数据同步工作。

要安装hugo-aligolia我们需要先确保我们已经安装了 npm 或者 yarn 包管理工具。

安装命令行

$ npm install hugo-algolia -g

安装完成后，在 hugo 生产的静态页面的根目录下面新建一个 config.yaml 的文件 (和 config.toml 同级)，然后在config.yaml文件中指定 Algolia相关的 API 数据。

---
baseURL: "/"
languageCode:  "zh-cn"
title: "易道非 blog"
theme: "flybirdNext"
description: "道可道，非恒道"
 


algolia:
  index: "github-yidaofeicnBlog"
  key: "d588f86bb333c536c79e3dbcff9b3bf6"
  appID: "NYC6MHRISW"
---

API 相关数据可以前往 dashboard 的 API Keys查看，注意上面的key是 Admin API Key。

配置完成以后，在根目录下面执行下面的命令：

$ hugo-algolia -s
JSON index file was created in public/algolia.json
{ updatedAt: '2018-11-19T05:54:57.155Z', taskID: 992973371 }

执行完以上命令后可以在 public 目录下看到已经生成一个algolia.json的文件。另外在 dashboard->Indices 中看到相关的数据：

如果某篇文章不想被索引的话，我们只需要在文件的最前面设置 index 参数为 false 即可，hugo-algolia插件在索引的过程中会自动跳过它。

JavaScript 代码

<script type="text/javascript">
$(document).ready(function() {
        establishSearch("#search-input");
});


function establishSearch(searchInputSelector ){
    //config.toml 传递值到页面，Jquery获取相关的值
    var   AppID=$("#algoliaData").data("appid");
    var SearchKey=$("#algoliaData").data("searchkey");
    var initindex=$("#algoliaData").data("initindex");

    /* 单页面认证   
    var SearchKey="7f131a263a8820645ee9fc65ed97b22f";
    var AppID="NYC6MHRISW";
    var initindex="github-yidaofeicnBlog";*/

    var client = algoliasearch(AppID, SearchKey);
    var index = client.initIndex(initindex);


    $(searchInputSelector).autocomplete({ hint: false }, [
      {
        source: $.fn.autocomplete.sources.hits(index, { hitsPerPage: 8 }),
        displayKey: 'name',
        templates: {
          suggestion: function(suggestion) {  
            return '<span>' + '<a href="/' + suggestion.uri.toLowerCase() + '">' +
                suggestion._highlightResult.title.value + '</a></span>';
             
          }
        }
      }
    ]).on('autocomplete:selected', function(event, suggestion, dataset) {
        console.log(suggestion, dataset);
    }); 
}
</script>

以上

autocomplete 自动 HTML 代码美化

调用 autocomplete.js 会自行生成相关的 HTML 代码，链接 https://github.com/algolia/autocomplete.js#look-and-feel

代码如下：

<span class="aa-dropdown-menu{{#datasets}} aa-{{'with' or 'without'}}-{{name}}{{/datasets}}">
  {{#datasets}}
    <div class="aa-dataset-{{name}}">
      {{{header}}}
      <span class="aa-suggestions">
        {{#suggestions}}
          <div class="aa-suggestion">{{{suggestion}}}</div>
        {{/suggestions}}
        {{^suggestions}}
          {{{empty}}}
        {{/suggestions}}
      </span>
      {{{footer}}}
    </div>
  {{/datasets}}
  {{empty}}
</span>
