基础软件

关于基础软件，我的定义是：

  1. 不涉及任何专业领域软件，如编程 「Eclipse」，毕竟大家的工作环境各有差异！
  2. 不涉及任何复杂的安装过程，开箱即可使用

简单来讲就是通过这些软件你能够正常的上网、看视频、看PDF、聊天、写 Word/Excel、分享文件就可以了！

推荐详答

安全类-火绒

  1. 火绒好是好，但是请不要吹过头了：文章主要内容：
    • 病毒查杀率低？
    • 对微软系统新技术支持不好？
    • 火绒对 HiDPI 屏幕的支持？
    • 火绒的定位是一款辅助安全软件？
  2. [感谢您的质疑，容我们解释一二-火绒针对上文的回复]：文章主要内容：
    • 病毒查杀率低
      • 火绒只重视对「活着的病毒」进行查杀
      • 自研的「通用脱壳」引擎——更加注重实际情况
      • 专注于打磨本地引擎
    • 对微软系统新技术支持不好？
      • 「对产品有用才用」——不管实质怎样，此理论非常赞同，一切为用而用，而非专注于器
    • 火绒对 HiDPI 屏幕的支持？
      • 下个版本（5.0版本）解决
    • 火绒的定位是一款辅助安全软件？
      • 以整体的终端安全解决方案为使命，并以卡巴斯基、ESET等国外最优秀产品作为比较对象
  3. 火绒关停流量业务，我找马刚聊了聊：文章主要内容：
    • 流量业务如何「作恶」
    • 不作恶：「但是，说到我们的品牌，实在惭愧。我不认为我们做了什么好事，我们只是没做坏事。我们没有太高的情操觉悟，只想尽力不作恶」
  4. 我为何不再使用“专注、纯粹”的火绒安全软件？：火绒没法检查出「相关病毒」
