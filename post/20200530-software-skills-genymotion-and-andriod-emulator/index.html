Andriod-Genymotion 与 Andriod 模拟器

2020年05月30日
 read
   Categories: 软件技巧

关于 Andirod 模拟器大家耳熟能详的大约有：蓝叠、夜神、雷电……，但是其实只有两大流派：Bluestacks 和 Virutalbox，其代表的产品如下：

  • Bluestacks：靠谱助手、新浪手游助手
  • Virutalbox：海马玩、逍遥安卓、夜神、ITools、Genymotion

在经历了「Bluestacks」启动卡死、「Nox」无法装微信等、「雷电」广告过多之后我选择了「Genymotion」，虽然前期配置麻烦，但是真的是用的很溜！

Tips：本次所有数据是基于Windows10操作系统、Genymotion3.0.4版本、VirtualBox 5.2.18 r124319 (Qt5.6.2)版本、Andriod SDK 8

ADB 命令安装

此安装方法来自官方032_Deploying_an_application.html#adb-method以及解决genymotion-arm-translation.zip无法拖拽安装的问题

  1. 打开 「Windows」 命令行，切换到「ADB 工具」工具目录之下

  2. 将「Genymotion-ARM-Translation」移动到「Google Pixel」之下

    adb push D:\APK\Genymotion-ARM-Translation_for_8.0.zip /sdcard/Download/

  3. 刷入程序

    adb shell flash-archive.sh /sdcard/Download/Genymotion-ARM-Translation_for_8.0.zip

  4. 重启系统

    adb reboot

  5. 确认是否刷入

    adb shell getprop ro.product.cpu.abilist

    返回值应该为：x86,armeabi-v7a,armeabi，如果不是则说明安装失败
