幸福进化俱乐部—年度计划课程笔记3—补充：个人成长定位系统

2018年12月14日
 read
   Categories: 人生管理课程

本文为幸福进化俱乐部—年度计划课程笔记 2—定位、选择、布局 的补充，主要内容来自《多贝-易仁永澄—进化路上的迷失与定位》课程笔记！

主要内容围绕以下图展开：

思考角度

课程从以下四个方面思考人生关键节点和过程

  1. 心智水平
  2. 好奇心&精力
  3. 自我&世界的关系
  4. 认知&行动

相关状态以及策略

0状态

0状态：刚出生的婴儿
趋0过程：也就是怀孕的过程，这里是有关胎教的部分，省略带过

A状态

主要特点：

  1. 七八岁，开始认生，开始有自己的意识，
  2. 精力和好奇心旺盛，各种问题，也会问自己从哪里来
  3. 自我与世界的联系不多
  4. 认知和行动还没有

趋A过程：
此时家庭教育最重要，需要父母有系统的育儿观念，相信孩子是有灵性的，这个部分也略过

B状态：

心智力量开启，在无意识中积累

主要特点：

  1. 心智开启的瞬间，是个关键点
  2. 好奇心很强
  3. 自我不断膨胀，如图中的线，线是用来扩展自己心智力量的方法：体验、试错、观察、阅读学习、思考、实践
  4. 心智开启，有了认知，行动上的能力有一些，但缺乏能力

趋B过程：
这个时长可以很长，短的十几年，长的一辈子，只有10%的人能到达B状态，也就是心智开启的状态

怎样做可以缩短孩子的趋B过程呢：

  1. 家庭教育：锻炼基本素养，系统的教育理念，指引他的好奇心
  2. 学校教育：大学前的学校教育有所欠缺，尤其是阅读能力、实践能力需要加以培养
  3. 社会教育：快速开启心智的方法，就是社会教育

PS：过了B点后该怎样快速成长呢，没有捷径，必须通过不断的学习、思考、实践

C状态（冲突状态）

内心充满焦虑感，什么事都不知道做不做得好，要不要做

主要特点：

  1. 心智水平快速增加，因为心智开启
  2. 好奇心与精力之间出现不可调和的矛盾，精力不够用（图中往里缩的黄线），好奇心却越来越多（图中往外扩展的虚线）
  3. 自我在扩大，对自我认识更加清晰
  4. 认知水平和行动能力之间的差距是最大的，知行最不合一的时候

趋C过程：
认知不断扩大，但焦虑感也在增大：

  1. 发现世界有这么大，我自己这么小，努力了这就久还是离世界很远，好奇心与个人精力之间的矛盾。
  2. 自我与真实的自我之间的差别

趋C过程问题解答：

  1. 原则：积累、为学日益
  2. 经历时长是可以通过一些方法缩短的，如：
    • 好的方法，如个人管理系统构建
    • 理念新不新，适合当下的
    • 工具快不快，工具快，积累效率高
    • 有没有加入社会化学习
    • 时间付出的够不够
    • 身体机能能否支撑
  3. 本阶段关键词：先抄后超
  4. 先提高认知再考虑行动
  5. 关于Mentor（导师）：真心对待，愿意帮助的良师益友。
    • 必须是自己真正的朋友
    • 在某一个领域超越自己，无所谓什么领域，越多越好

PS：到了C状态以后，怎样调和好奇心与精力的矛盾呢？两个办法：

  1. 把认知降下去（当然此方法是不可取的，好不容易进步，你愿意又退步回去吗）
  2. 把行动升上来：找一个点突破，持续的积累不可少

D状态（专注状态）：

自我与世界已经有一个点真正接触，找到自己的意义所在

  1. 心智水平空前提高，并急速积累
  2. 好奇心取向与行动取向是一致的
  3. 我和世界已经有了沟通和联系
  4. 认知和行动能力都很强，可以达到知行合一的状态

趋D过程：

  1. 原则：为道日损、选择与放弃（生活就是做减法的艺术）。
  2. 道是什么？答案是什么道都行，只要能达到和世界交流的目的，但也要考虑自己的天赋，考虑自己的资源，考虑导师的问题（这个阶段的mentor不一样了，应该是你那条路上指引你走路的） so选择自己的路，“走自己的路，让别人去说吧”
  3. 如何做到损：
    • 是从心智成长的三要素入手
      • 学习方面：和自己的道无关的，都舍弃掉。（如人脉管理需要考究，不需要每天关注）。读三类书：看让心智成熟的书、看专业书、用耳朵听，更高效。
      • 思考方面：看到的素材不能全部放到笔记里，最好是放经过内化的笔记
      • 行动方面：做可以提高自己的事，有基础的地方去做突破，做一些提高掌控能力的事
    • 是从个人管理系统上损（博客1479文章），从前置判断、知识高度分类以及建立个人行动管理系统这三个部分都可以。
  4. 怎样兼顾好奇心和专注：
    • 精深训练：每天用2小时47分钟的时间做一件事，其他时间可以满足好奇心
    • 对100个梦想清单说no，这个时候，不做清单，或者可以做100个不做的事清单
    • 虽然看似慢，但实际上是很快的。这是在打造核心竞争力，速度快、效果好、质量高，专业产生价值
    • 年计划的意义开始呈现，因为不再被好奇心指引
    • 增加问题管理系统、思考管理系统
    • 导师发挥关键意义

圆融状态：

在这个状态，就可以关注世界的各个方面，与世界圆融

圆融阶段：

  • 想办法解决固定思维。
  • 从专注状态一定能到圆融状态么？当然不一定。
  • 这里有没有反复和曲折？肯定有，所以很难达到圆融状态。
  • 这个阶段老人家可以帮助到自己，老子、释迦牟尼等老人家
