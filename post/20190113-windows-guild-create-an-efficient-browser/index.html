浏览器选择

上网的第一步就是选择浏览器，浏览器从国度上可以分为国内与国外，我的选择是国外，至于为什么如下：

  1. 就算不谈政治，国内互联网公司无耻程度超乎大家的想象
    • 中科院报告：360产品存在三大隐私安全问题：360将「用户访问的网址、网址对应的 cookie、用户在地址栏输入的内容」等信息上传到公司服务器
    • 百度竞价排名思考 - 血友病
  2. 国内绝大部分浏览器基本上都是「壳」，无自己的内核

在国外的浏览器我选择的是「Firefox」或「Chrome」，主要原因在于扩展多，使用人多，可定制化程度高

关于两者之间的比较，我参考弃用 Chrome 改用 Firefox 的几点理由——关于 Chrome 69 隐私丑闻的随想整理如下：

  1. 隐私性
    Mozilla 是一家非营利机构而 Google 是一家商业公司
    • Google 的商业模式注定了它收集隐私的偏好：Google 利润 90% 以上是来自于广告收入
    • Chrome 缺乏「网站中立性」：Chrome 69 版本严重的隐私漏洞——《Chrome 69 清空 Cookies 会保留 Google 的 Cookies @ Solidot》
    • Chrome 可能会与 Google 的服务暗中勾搭，以收集隐私
    • Firefox 是完全开源的，而 Chrome 是基于 Chromium 增加了一些闭源模块
  2. 安全性
    • 小众 VS 大众，可以类比 Windows 病毒远远超出 Linux、Mac
    • Rust VS C++，前者相比后者安全性更高
    • 「自带 CA 证书」 VS 「使用系统 CA 证书」：自带的 CA 证书不会受到操作系统及其厂商的影响
  3. Tor 社区对 Firefox 的加持

加上在中国后者的无需翻墙就可以轻松使用，所以后面我也会逐步迁移到「Firefox」之上！

油猴 Tampermonkey

为什么会将此单独列出呢，实质上他就相当于一个平台，插件非常多，在一定程度上可以代替扩展，并且 Firefox 与 Chrome 都有这个插件，所以将其单独成章节！

油猴 Tampermonkey 的官方网址是：https://tampermonkey.net/

油猴默认是没有安装任何脚本的，你可以通过设置页面的「已安装脚本 - 获取脚本…」来下载各种脚本。

比较常用的下载渠道有三个：

  • Greasy Fork：支持中文，按照今日安装、总安装数、得分、创建日期等的排序方式给出脚本列表，可按脚本生效的网站过滤，每一脚本都有中文介绍，并且详细列出了作者、安装数、更新日期日志、安装使用截图、兼容性、应用到、代码查看等信息描述。
  • OpenUserJS、Userscripts Mirror：这两个网站都没有提供中文界面，Userscripts Mirror 已经停止了更新，用户可以在这个站点找到历史资源。

找到需要的脚本后，会在介绍页面看到安装（install）按钮，点击下载脚本后会自动跳转到安装界面，再点击安装就可以享用脚本了。
