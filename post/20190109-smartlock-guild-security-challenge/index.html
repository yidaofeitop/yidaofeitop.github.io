技术开锁——小黑盒事件

@滕腾 小黑盒子事件说句实在话是“一粒老鼠屎坏了一锅粥”的最佳体现，对于指纹锁的销售确实是一次比较大的困难期，但是从另外 一个层面来讲也促进了指纹锁行业的良性发展——指纹锁曝光度增加、低质量厂家清场、行业标准进度加快等，这是我对于此次事件的一个看法！

但是也请大家理性的看待相关的宣传报答，并不是说所有的指纹锁都有此现象，有很多的厂家都发表了相关的测试视频：

技术开锁——假指纹

好奇实验室 第二季：指纹锁真的能被轻易破解吗? 可以很清晰的看到复制指纹并没有想象中的那么简单——有谁会傻逼到给别人按住3分钟复制指纹 :😄:

其他破解方式

破解方式 应对措施
技术开锁——破解钥匙 C 级别锁芯
暴力开锁——破坏前面板 真插芯+电子锁体+逻辑处理模块后置（王力智能锁方案）
技术开锁——猫眼开锁 防猫眼开锁技术
技术开锁——卡住主方钢转动 方钢套铸造为原型或外加一个圆环、嵌入门面内

另外纠正一个错误的观念：

破解 IC 卡，指纹锁中的 IC 实质上就是卡的ID输入指纹锁内部存储起来用于对比的一串文字，就相当于一个把钥匙，卡本身没有存储数据的功能无需像公交卡等破解，IC 卡只有遗落的风险—》指纹锁的便捷性（可删除相关的卡）
