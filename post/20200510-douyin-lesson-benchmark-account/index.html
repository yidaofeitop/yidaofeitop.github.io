抖音-理论部分-6：对标账号

2020年05月10日
 read
   Categories: 短视频

在确定完成抖音-理论部分-5：抖音之定位之后有一个非常关键的点——「寻找对标账号」，尤其是对于抖音菜鸟来说尤为重要！

寻找对标账号的主要目的如下：

  1. 通过对对标账号呈现内容主题的研究，能够进一步的将自己的思路打开、条理更加的清晰
  2. 通过对对标账号视频结构的研究，能够对同类型的主题如何写脚本等有一定的画面感
  3. 通过对对标账号的标题研究，能够对如何取标题以及如何选取热门话题的技巧有一定的掌握
  4. ……

可以说通过对对标账号的研究基本上能够对视频创作会有一个比较全面的了解！

本文主要分为以下内容：

  1. 对标账号的基本特征
  2. 如何找到对标账号
  3. 如何拆解对标账号
  4. 验证对标账号方法

对标账号数据指标

对标账号的基本要求如下：

  1. 总粉丝数超过50万
  2. 单个视频涨粉数据高。

前者其实是说明账号整体运营能力好，后者说明账号的运营效率高

对标账号表格

账户名 账号ID 视频个数 视频主题 表达形式 粉丝总量 近3个月粉丝数量 1ST 视频数据 2nd 视频数据 3rd 视频数据

如何找到对标账号

  1. 向前辈高人请教
  2. 利用抖音搜索栏直接搜索 在抖音搜索栏直接搜索我们希望了解的行业或者关键词，会搜索到非常多的优质大号
  3. 抖音关注某个账号，系统会自动推送同类型账号
  4. 利用新榜等第三方平台，查询有哪些优质账号
    • 百度搜索「抖音大 V」等关键词
    • 查询抖音分类 KOL 榜单小程序（TooBigData、抖大大）

Tips：竭尽全力去寻找同平台的对标账号，你找100对标账号比找10个绝对会物超所值

如何拆解对标账号

  1. 将搜集的账号按照主题内容或表现形式大致的分一下类，然后在分类内部做对比

  2. 利用视频拆解表完成对视频的拆解

    账户名 账号ID 视频标题 视频主题 时长 脚本结构 拍摄手法 视频剪辑包装 留言区 播放数据

  3. 分析账号的爆款选题和其普通选题的差异之处

验证对标账号方法

  • 减少变量，先做竞品爆款视频的同类选题
    最开始的几个视频，要无限逼近竞品的拍摄形式、演员状态、选题和结构方式。一般10个视频左右，就能看出来数据是否OK了
  • 方法论验证OK，大胆创作新选题，拍摄形式、演员结构等都要创新
    • 同类账号做过的爆款选题
    • 同领域的公众号/知乎/微博热门话题
