Windows-如何在 Win7 原生安装U盘中加入 USB3.0 的驱动

2020年10月02日
 read
   Categories: 软件技巧

此问题主要是发生在一个 HP 台式机上，原有的系统是 WIndows 10 系统，引导采用的 UEFI ，由于卡顿所以需要改装成 Windows 7，在安装过程中出现一个非常有意思的情况：微PE界面可 正常使用鼠标键盘、进入 Windows 安装程序之后全部失灵，首先是怀疑由于改装 Windows7 导致主板不匹配，整整浪费了一个上午，后续查询才是 原生Windows7系统不包含 USB3.0 的驱动

本文主要是记录如何做成一个 Windows7 含 USB3.0 的安装介质

Step1：使用软碟通（UltraISO）制作 Windows7 原版安装盘：可参考：使用UltraISO（软碟通）制作U盘启动盘完整教程

Step2：下载Windows USB Installation Tool

Step3：打开解压缩后生成的「 WindowsImageTool」文件夹；鼠标右击「 WindowsImageTool.exe」，选择「以管理员身份运行」；

Step4：按照下图箭头指示和文字说明操作；

Step4：制作过程需要数分钟，看到「Creating Windows USB Installation Disk is complete」提示时代表已操作完成，点击「Close」关闭工具软件。
