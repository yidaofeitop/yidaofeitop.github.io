MacOS-Mac 电脑U盘重装系统

2020年09月22日
 read
   Categories: 软件技巧

不知道有多少人出现过这种情况：唯一的 Mac 电脑系统无法进入，网络安装选项也出现问题，到网上找相关的镜像安装制作 PE 却无此启动项，:😢:

可能是如下的原因：

  1. 镜像选用的不是「非引导镜像」
  2. 镜像选用的版本不对

第二个原因系统会有比较明显提示，而第一个原因则仅仅是「无此启动项目」！

本文仅记录在帮同事搞定 Mac 电脑双系统中的一些细节点，如想要安装的步骤大家可以网上查询相关资料！

引导镜像 VS 镜像

正如制作macOS系统dmg包及iso可引导镜像所说：

实际上网上少有的提及macOS iso制作的，都是啰里啰嗦的搬上了一个博主的教程（需要近十条），而没有去分析其原理并加以简化。

这点深有感触，之所以我使用 U 盘安装 MacOS 使用了近乎2个通宵的时间，很大程度上就是因为网络上很多文章都是「模仿」，没有交代前因后果，不谈及重点！

Mac 安装系统一般以下方式：

  • Mac 系统正常，正常更新、升级
  • 利用 U 盘等外部介质安装

前者主要的问题是如何下载到想要的版本，可以参考此文章macOS完整安装包下载方法

后者则涉及 可引导镜像（含安装程序）、Etcher 等 U 盘启动盘制作工具！

关于「可引导镜像制作」是怎样的一个概念可以参考制作macOS系统dmg包及iso可引导镜像-DMG镜像包的制作

  1. 多系统可参考macOS安装/Winpe多分区工具盘制作
  2. 单系统可参考如何在 Windows 下制作苹果 macOS 启动盘

双系统安装

参考MacBook安装Windows10系统完整过程

注意其中如下关键词：

在Windows10刚刚安装好之后，驱动并没有安装完毕，因此是无线网卡启动不了，无法接入Wifi。此时，进入Windows10的 OSXRESERVED 分区，这通常是 D:\ 驱动器。打开 OSXRESERVED 分区中的“BootCamp”文件夹，然后运行“Setup”文件。安装完毕后，驱动也就装好了，WiFi可以连接了。会发现OSXRESERVED分区不见了。
