Sublime教程： 第二章 Markdown 编辑环境搭建

2020年01月18日
 read
   Categories: 软件技巧

作为一个 Markdown 重度使用者，既然使用上了 sublime 那么 Markdown 绝对是第一个要安装的插件，在这里分享一下我 整个 Markdown 工作环境搭建的过程！

本文主要内容分为以下方面：

  • 插件安装及配置
  • 安装问题

如何安装插件：https://packagecontrol.io/installation

  • MarkdownEditing：支持Markdown语法高亮；支持 Github Favored Markdown 语法；自带3个主题

  • OmniMarkupPreviewer：实时在浏览器中预览，快捷键如下：

    • Ctrl+Alt+O: Preview Markup in Browser
    • Ctrl+Alt+X: Export Markup as HTML
    • Ctrl+Alt+C: Copy Markup as HTML

MarkdownEditing 配置 在「Preference——>Package Settings ->Markdown Editing->Markdown Setting: User」中修改：

{
    "extensions":["md"], 
   "color_scheme": "Packages/MarkdownEditing/MarkdownEditor-ArcDark.tmTheme",  // 修改主题
    "line_padding_top": 4,
    "line_padding_bottom": 4,
    "tab_size": 4,
    "translate_tabs_to_spaces": true,
    "draw_centered": false,
    "word_wrap": true,
    "wrap_width": 100,// 每行字符数上限 
    "rulers": []
}
