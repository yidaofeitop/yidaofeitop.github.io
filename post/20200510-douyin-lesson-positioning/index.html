抖音-理论部分-5：抖音之定位

2020年05月10日
 read
   Categories: 短视频

前面4篇文章主要是介绍抖音的一些基础认识，从本文开始就开始进入实操环节！

在写这篇文章之前我看了不少的「抖音韭菜课程」，绝大部分都是直接开始「什么头像设计……」，少部分的「抖音定位：内容定位……」也是讲解的很浅

我从自身的项目实践出发讲讲抖音最为重要的一堂课程之二「抖音定位」，本文主要内容为以下部分：

  • 为什么要做定位？
  • 定位分为哪些定位？
  • 内容垂直如何做？
  • 抖音有哪些变现方式？
  • 还有哪些因素需要考虑的？

Tips：：我的建议是大家将抖音作为一个商业计划来看到真正的写一份「商业计划书」，如果你对每个时间点做些什么怎么做要达到怎样的效果有一个清醒的认识的话基本上就能够对时间与金钱做到最大化的 利用！

为什么要做定位

  • 视频内容总是在变，今天拍蜂蜜、明天拍服饰、后天拍土鸡蛋……
  • 账号标签为搞笑、美女……完全不是自己想要的
  • 粉丝画像是6-17岁为主、男性为主，但是我的变现是通过抖音直播买服饰……
  • 我的产品是xx家电经销商，我本想通过抖音买点货但是现在居然没有一个访问量 :😢:
  • 我的粉丝量增长很慢，究竟是怎么回事呢？我是不是要转型一下娱乐内容来增加粉丝
  • 我自己能够现在做的内容已经全部做完了，但是感觉还有一些东西没讲……
  • ……

不知道大家是否有以上的问题？——做着做着就发现完全不是自己想的那样，感觉到处都是问题，又不知道从何解决？

这个问题就是大家在做之前没有做好相关的定位：自己要做什么内容会吸引怎么样的粉丝能买什么产品？

表面上这个问题比较简单，但是如果没有进行深入的思考的话最终也会「死」的不明不白！

以下就开始正式的说明如何做好定位！

定位分为哪些定位

思考角度

借助营销领域的定位，我们来分解一下短视频创作领域的定位应该是怎样的？

  1. 领域定位：了解自己或者组织在市场上怎样的一个位置？头部品牌、二线品牌或者其他？
  2. 产品定位：产品属于哪个档次，高端、中端、小而美路线，相对应的定价范围是怎样的， 产品对标的是哪类客户
  3. 用户定位：中高低客户的爱好是怎样的？消费能力怎么样……
  4. 内容定位：基于以上分析客户喜欢怎样的内容，我们能够生产什么样的内容呢？生产的效率怎么样？

以上的定位方式大家是否理解呢？如不能理解的话我们我们从变现出发来倒推：

  1. 理顺你现在手头有哪些资源或者即将拥有的资源
  2. 调查你的资源的情况
    • 市场情况调查：品牌真实情况、线下销售模式、定价及客户群体、抖音同类产品销售情况（定价、销量）
    • 梳理资源特点：资源本身特点及衍生的概念、对相关特点、概念进行分类
  3. 梳理客户情况
    • 针对资源本身特点及衍生的概念来描述对应的用户画像
    • 用户针对资源的决策流程是怎样的？
    • 资源本身特点及衍生的概念与客户的痛点关联性
  4. 梳理内容主题
    • 针对客户画像及痛点应该输出哪些内容
    • 内容可以分解哪些主题

定位案例

  1. 本身资源：
    • 产品：云南野生蜂蜜（年产2吨）、土鸡蛋、彝族民族服饰
    • 组织：采集蜂蜜团队（3个人）、土特产店营业执照、服饰店（个人）……
  2. 资源情况
    • 无品牌，线下销售模式……
    • 资源特点：野生蜂蜜，相关联概念：健康保健、纯天源、好奇心理
  3. 客户情况
    • 用户画像：
      • 极其注重生活质量，只要品质好价钱无所谓
      • 注重生活品质但是也需要价格有优势
    • 客户决策因素
      • 是否是真蜂蜜、真野生蜂蜜？
      • 这蜂蜜相当于其他的蜂蜜优势在哪里？
      • 蜂蜜如何使用能够最大化？
      • 价格相比线下有哪些优势？
      • 售后怎样解决？
  4. 梳理内容主题
    • 蜂蜜Vlog—>养蜂技巧（参考「蜜蜂司令」）、采蜂蜜Vlog（景洪阿铁野生蜂蜜）……
    • 如何选购蜂蜜：蜂蜜种类、蜂蜜差异、如何选购蜂蜜
    • 如何使用蜂蜜：结合蜂蜜的各种用法……
    • 其他更多的土特产

定位考虑清楚之后就相当于你能够利用现有的资源输出多个主题内容吸引目标客户，后面要解决的问题就是怎样把内容做到垂直，最大化的利用内容来吸引目标客户！

内容垂直如何做

内容垂直怎样做，我们团队在2020年的时候梳理出一张如下的图：

后来在实践的过程中发现真正落地的实质上只需要做好以下几个部分：

  1. 明确自己主领域 即「行业垂直」，这个方面大家很容易与「领域定位」或者「内容定位」、「统一主题思想」混淆，觉得三者是差不多的，实质上差异很大。

    • 「领域定位」主要是讲的是个人或组织在社会上的关系，注重的是「外部的事情」
    • 「内容定位」主要是从「领域定位」……「客户定位」说明我们需要产生怎样的东西
    • 「行业垂直」则是我们基于某个点进行大范围的思考，比如说「野生蜂蜜」可以作为行业垂直-「蜂蜜垂直」，但是如果再扩展一点的话是不是能够将哪些内容不够丰满的土特产吸纳过来，那样的话我们的行业垂直是不是应该转为「土特产垂直」

  2. 统一主题思想
    利用抖音的合集功能将每个主题分类管理起来，做到内容垂直度高，不如说「如何培养蜂王」、「蜂箱如何分箱」……归为「养蜂小知识」

  3. 确认基础人设+外展人设

    • 基础人设是整个账号的人设，比如说「土特产搬运工」
    • 外展人设一般而言是说对基础人设的一个补充，让基础人设更加的丰满，如：多次在视频中用民族服饰 出场

  4. 统一基础部分
    头像、背景、昵称三部分协调统一；视频封面协调统一

抖音有哪些变现方式

关于变现部分实质上应该在思考「定位」部分同步考虑，因为无变现就没有利润，没利润的事情一般来说都 不长久，所以在开始之前最好就把变现考虑清楚！主要考虑以下方面：

  • 平台变现的最低要求是什么？比如商品橱窗是：1000个粉丝+10个作品
  • 你预期的最低变现是怎样的？多长时间投入多少产出多少？
  • 你变现的渠道是否已经调查过？如直播带货有没有调查过抖音平台同类型的产品价位销量……是否有加速渠道……
  • 如果变现未完成那么你将会损失什么？得到什么？能够采取哪些措施将损失降低到最低？
  • ……

还有哪些因素需要考虑的

  • 人员配置是怎样的？哪个部分是重点部分？
  • 财务是否需要做一定的防火墙？
  • 其他部门的支持力度是怎样的？能够提供哪些支持或者有哪些阻碍？是否要做好利益的规划？
