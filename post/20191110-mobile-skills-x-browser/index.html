Andriod-X浏览器基础设置

2019年11月10日
 read
   Categories: 软件技巧

前天晚上母亲忽然来微信电话说「找不到百度入口」，后来才发现是跳转的链接完全不对

后续继续分析发现是因为我在购入之时就刷入了「MIUI国际版本」所以导致浏览器也是国外的界面 :😢:

在母亲无意中解决了之后想想「为什么我自己使用的 X浏览器写一个简单版的教程出来以便大家更好的上网」

本章无意去比较那个浏览器更加的好，因为每个人的需求都不一样，我选择 「X-浏览器」的原因是在于

  • 非常简洁，主页可以设置为只有一个搜索引擎的入口，类似 Google 搜索的入口
  • 广告拦截 非常给力，相信大家在移动端对于广告应该是深有体会，而该浏览器在这方面可以说超越了绝大部分浏览器
  • 可扩展油猴脚本，定制化非常高
  • 最后的最后就是体积非常小，仅仅只有1M不到
