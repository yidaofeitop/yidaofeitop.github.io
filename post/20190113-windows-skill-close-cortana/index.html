Windows 技巧7-保护隐私-关闭微软小娜 Cortana

2019年01月13日
 read
   Categories: windows 教程

一直都知道微软小娜是个人隐私泄露的最大地方，但是由于一直在写其他的文章，所以直到今天才动手完成此项！

本文内容全部来自Win10 如何关闭微软小娜 Cortana，本文仅作内容的调整！

以下将介绍三种方法关闭微软小娜 Cortana

  1. 使用 Dism++关闭 Cortana
  2. 组策略禁用 Cortana——Windows 10 专业版
  3. 注册表关闭 Cortana——Windows 10 家庭版

使用 Dism++关闭 Cortana

下载 Dism++ 并解压，运行 dism++.exe 启动程序。

系统优化 -> 开始菜单以及Windows体验 -> 关闭多嘴的小娜。

组策略禁用 Cortana——Windows 10 专业版

「win + R」打开运行，输入 gpedit.msc 回车打开组策略。

左边栏选择 计算机配置 -> 管理模版 -> Windows组件 -> 搜索，双击 「允许使用 Cortana」 -> 已禁用 -> 确定。

注册表关闭 Cortana——Windows 10 家庭版

「win + R」打开运行，输入 regedit 回车打开注册表。

地址栏定位到：HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\Windows Search

右键 「Windows Search」 新建 「DWORD (32位)」 值，并命名为 「AllowCortana」，然后双击 「AllowCortana」，将 「数值数据」 设置为 0，确定。
