打印机-如何设置远程扫描

2019年11月03日
 read
   Categories: 软件技巧

之所以记录如此简单的东西主要是在今年7月份查询很多资料但是仍旧无法配置好打印机的远程扫描，身为一个曾经的程序员深感惭愧 :😢:

本文内容大致分为三个部分，全部以图片的形式呈现：

  1. 第一部分：电脑端安装打印机驱动
  2. 第二部分：电脑端配置远程扫描文件
  3. 第三部分：打印机配置扫描

Tips：电脑系统 Windows 7 旗舰版，打印机型号为 RICHO MP4001，具体到其他型号可能稍有差异，大家理解相关步骤的主要目的即可

电脑端安装打印机驱动

电脑端配置远程扫描文件

本部分大致可以分为两部分：共享文件配置、共享网络配置

第一部分 共享文件配置

此处是一个比较重要的点，很多的朋友可能会犯以下的错误：a.文件名采用中文（可能会导致打印机乱码）b.文件访问权限 「everyone」的权限仅仅开通「写或读」

第二部分 共享网络配置

Tips：一般而言系统右下角「网络标志」右键点击「打开网络和Internet设置」

打印机配置扫描

此步骤相主要是定位远程电脑的位置，MP4001 具体的步骤为：

  1. 按下「初步设定/计数器」
  2. 点击「系统设置」->「管理员工具」->「通讯簿管理」->「新编入」
  3. 输入「名称」与「主要显示」
  4. 点击「文件夹」->选择「SMB」->输入路径（路径格式为：「\电脑名称\共享文件名称」）

输入访问「电脑端共享文件」的账号，由于前面我们选择了「关闭密码保护」，所以随意输入信息即可，一旦提示「连通」即表明配置成功

点位扫描功能，看是否真正的设置成功:)
