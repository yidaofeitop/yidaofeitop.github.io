指纹锁品牌与实力

在行内有一句话：“指纹锁没有品牌”，“指纹锁没有品牌”，“指纹锁没有品牌”！

之所以如此说是因为指纹锁真正开始大规模的落地的时间点是在2017年初，大家半斤对八两都没有将自己的品牌深入的广大的消费这心中！

实力方面第一梯次的为传统锁具知名品牌如如樱花、名门类等；第二梯次的为安防领域压铸厂家转型与以前的次等的传统锁具厂家，在这方面大家搜索相关品牌的背后生产商的历史就可以清晰的了解其相关的实力！

我仅以自己深入接触的品牌说明推荐（至少拆解过该厂的某个产品或参观过生产基地）：

  1. 金指码：2018年6月 长沙居然之家朋友分销产品——产品外观处理工艺最佳、电子锁体、稳定性好
  2. 名门：2017年11月 重庆建博展与当地代理商接触，试用其产品——传统五金锁具巨头，产品简约大方，稳定性好
  3. 樱花：2017年9月 参观生产基地，传统五金锁具巨头，结构简单实用，稳定性好
  4. 普罗巴克：2017年初 参观生产基地，2017年前智能锁行业人才最为集中地方，锁具外观普通，稳定性好

关于其他未深入接触的品牌就不予以评估，但是要纠正大家1个认识：指纹锁技术在近2年之内并没有很大的突破，基本上就是传统的安防领域过剩技术的转移，所以没必要过多去强调国内品牌还是国内品牌！

典型的代表三星在我看来就是一个笑话，内部结构过于复杂，售后成本高、外壳为塑料壳能 hold 住什么！

综上，如果你仅在乎品牌与售后，可以从上面的品牌或者其他传统五金锁具巨头如汇泰龙的线下门店或淘宝购买相关，一般而言都不会被坑，阅读到此就可以去直接购买了！

但是如果你想要从2000多个品牌中的指纹锁找到性价比极高的适合自己需求的指纹锁就请继续往下读！

如何选购一把适合自己的指纹锁

我们将首先从指纹锁的三大特性出发说明设计的整体细节，然后再对应相关的指纹锁关键部件，大家通过本章节应该可以很好的了解一款指纹锁的要点是哪些，能够从关键点挑选出自己想要的特性组合选购出一把适合自己的指纹锁！

指纹锁特性

传统锁具的特性是“安全、稳定”，指纹锁是在传统锁具上增加了一个特性“便捷”！我们将从以上三大特性出发看看工程师在设计的过程的部分思路，从而了解为什么会如此设计！

指纹锁特性之一安全

我们从盗贼与工程师的角度出发来一一看如何破解与防御一把指纹锁：

盗贼 工程师
技术开锁——假指纹 请盗取用户指纹（难度不小）—>光学可破，半导体、指静脉不可—>
技术开锁——密码多次尝试 5次输入错误禁止输入5分钟，破解成本高；另联网锁可远程发送报警消息
技术开锁——卡 类比钥匙，卡删除（便捷）
电路攻击——小黑盒 软件代码容错处理，程序自动识别并处理外部干扰的错误信息，并加入相关防护
技术开锁——破解钥匙 C 级别锁芯/防撬报警
暴力开锁——破坏前面板 真插芯+电子锁体+逻辑处理模块后置（王力智能锁方案）/防撬报警
技术开锁——猫眼开锁 防猫眼开锁技术
技术开锁——卡住主方钢转动 方钢套铸造为原型或外加一个圆环、嵌入门面内
指纹锁特性之一稳定

我们主要从指纹锁可能发生的故障与安装难点出发，看有哪些不稳定点：

  1. 电机离合故障：电机不转、离合销脱落、离合回位不正
  2. 线路板故障：指纹无法识别、数字按键无效、耗电过快、无故重启、无故开关门
  3. 方钢滑动导致的前把手与锁体无任何的动力传递结构
  4. 使用一段时间后天地钩脱落，无法开门
  5. 电池无电

针对以上所有的可能的工程师是如何在设计智能锁以及一旦发生如何处理呢？

故障点 最佳设计 常规处理方式
电池无电 设定紧急供电接口 USB 供电
线路板故障 保留钥匙 钥匙开门，更换部件
电机离合故障-电机问题 优化结构、保留钥匙 钥匙开门，更换部件
电机离合故障-非电机问题 优化结构、6068正装锁体+真插芯 真插芯则钥匙开门，非则猫眼开锁或直接暴力开锁，更换部件
方钢滑动 优化安装细节、6068正装锁体+真插芯 真插芯则钥匙开门，非则猫眼开锁或直接暴力开锁，更换部件
天地钩脱落 优化安装细节、6068正装锁体+真插芯 或安装是正确处置天地钩 暴力破坏前面板，定位相关天地钩孔解决，更换部件
指纹锁特性之一便捷

指纹锁是传统锁具的一个升级，增加一个特性“便捷”，此特性的表现形式主要是在开门方式的多样化，我们列举一下市面上指纹锁所有的便捷特性：

  1. 指纹
  2. 密码
  3. 卡
  4. 机械钥匙
  5. 临时密码/周期密码/用户分组
  6. 蓝牙开锁/NFC 开锁
  7. APP开锁/微信开锁
  8. 智能家居联动

以上方式使用频率除去钥匙以外是按照逐步降低的排列的，关于如何在这方面取舍我在指纹锁安装日总结——流程/细节/选择重要已经充分的说明，不再一一的说明。

指纹锁关键部件

通过指纹锁特性的分析大家应该基本可以看出指纹锁的关键的部件为：线路板、电机离合结构、锁体，下面我将每个部件逐一的拆分给大家看：

锁体

之所以将锁体放在第一位是因为锁体一旦确定面板的结构与布局基本上就可以确定了，另外很多厂商的锁体都是外购的，也涉及供应链后续配合的一系列问题！

我将从锁体的规格参数、与锁体配合的锁芯等方面给大家一一分解！

  1. 锁体规格：现市面上流行的指纹锁锁体主要的规格有6068、6054、6052，我 推荐选用6068，主要的原因是：该规格能够适配绝大部分的门 且有葫芦头的位置用于安装真锁芯，与上面的「指纹锁特性之一——稳定」相对应理解
  2. 真锁芯还是假锁芯：详细分析请参考文章：认识智能锁 chapter 4 - 智能锁锁芯-真插芯 VS 假插芯,我 推荐选用真锁芯，锁最本质解决的需求是什么？安全 稳定，而假锁芯针对如电机离合出现故障（如离合销脱落）、安装失误未确认主方钢长度是否够导致的主方钢滑动脱落无能为力，只能猫眼开锁或破坏门锁开门:😢:
  3. 机械锁体还是电子锁体还是电动锁体
    • 机械锁体工业设计技术已完全成熟大规模生产化，获取渠道最广
    • 电子锁体稳定性有一定降低，天地钩适配性与机械锁体一致，成本有所提高，面板与锁体存在接口对应问题，采购途径受限于锁体厂家；但如果就个人而言是稳定性与安全性最佳平衡点的产品
    • 全自动锁体比电子锁体稳定性进一步降低，天地钩适配性最低，采购成本进一步抬高，与厂家对接接口更加复杂，完全受限于锁体厂家（所以很多是自主开发）

确定以上规格后可以通过以下判断标准进一步的判断锁体的质量：

  1. 锁体外壳是镀锌板还是 304 不锈钢？可通过表面观察判断，也可以使用磁体验证
  2. 锁体斜舌归为弹簧是否考虑弹簧陷入卡位后方？
  3. 主方钢与主锁舌传动结合部分是齿轮咬合传动还是卡位传动？
  4. 主锁舌与内部推杆是采用焊接方式还是采用铆钉方式？
  5. 圆柱形锁舌是实心还是中空？
  6. 左右互换的是否方便？
  7. ……

以上问题如果你是终端用户基本上只能验证1、体验3（手感是否很顺畅判断），如是渠道商或厂家则可直接拆解一个锁体了解，记得拍照即可！

PS：拆解过后的锁体如无完全把握请不要流入市场，后果很严重！后果很严重！后果很严重！轻者锁匠师傅上门150，重者只能破门而入！

电机离合结构

智能锁行业发展至今按照动力来源大致可以分为：半自动锁、全自动锁，按照离合结构的不同可以分为以下三种：

  1. 电机+离合结构：技术自2016年底开始已经模块化产业化，且经过时间的验证已经成熟，稳定性最佳，安装精确度低，难度中等（典型代表电子锁体）
  2. 电机+全自动锁芯结构：罗曼斯引领开发，技术爆发主要是在2017年开年时间段，技术成熟，稳定性较差，安装精确度高，难度最低； 关于稳定差的原因主要是以下3点（其中2、3点是自己长沙亲戚家亲身经历的锁）：
    • 天地钩卡涩、安装不平行均会导致电机转动半圈或少部分无法开门情况
    • 所有的动作依靠锁芯完成，而锁芯与面板有结合部分，一旦出现后面板对锁芯有较大的压迫力（一般为纵向），钥匙也无法开门
    • 电机供电电压较高，一旦无电，必须使用特点充电宝才能紧急供电
  3. 全自动锁体：行业内使用比较少，动力全部来自锁体内部电机驱动，然后再通过一系列的杠杆获取较大的力将锁舌弹出，安装要求门缝小、精准安装等，稳定性较差，安装精确度最高，难度中等

关于第一种电机离合结构可以访问《认识智能锁 chapter 2 - 离合结构补充——分离式离合设计》

关于第二种离合结构主要注意点是电机的技术参数，如电压；

关于第三种离合结构没有深入接触，不予追加说明！

更加详细内容请访问：认识智能锁 chapter 1 - 电子部分

线路板

关于线路板由于属于电子工程师与嵌入式工程师范畴，必须具有一定的专业知识才能深入的辨别优劣，本章节主要是从过去的一些生产经验与售后经验出发的一些方法：

  1. 前面板内部布线规范与用线情况—》我坚信注重细节的厂家产品质量一定会把控的好
  2. 线路板是否做防水、防尘……处理，光线照射下可看出是否有刷一层三防漆
  3. 线路板插线接口是否牢固、与线材的端口结合是否紧密、是否加固定粘合剂（非必要）
  4. 线路板整体布局是否合理？如是否有线被其他部件损坏的可能、走线是否顺畅……
  5. 线路板主板上的电子元器件是否足够的多？可以将一部分非常差的线路板排除
