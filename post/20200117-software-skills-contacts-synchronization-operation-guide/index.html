Android&IOS-通讯录同步操作指南

2020年01月17日
 read
   Categories: 软件技巧

现通用的通讯录导入方式主要分为以下两种：

  • 工具软件类：利用「和通讯」APP 同步：类似的软件还有 「QQ 同步助手」
  • 利用 VCF 跨平台文本+云端平台同步

和通讯录

和通讯录的相关链接：

  • 和通讯网站：http://pim.10086.cn/views/login.html?channel=www
  • 和通讯APP：http://pim.10086.cn/views/download.html?channel=www

请预先准备好通讯录的基本信息：用户名+手机号码

相关的步骤如下：

  1. 注册「 和通讯」账号，定位到操作主界面，点击「 导入导出」按钮

  2. 定位到「导入导出」按钮，点击「下载模板文件」

  3. 按照文件模板输入相关数据，点击「选择文件」

  4. 手机安装「和通讯」APP定位到「联系人」同步即可

VCF+云端平台

现在的手机基本上都有云端平台，可以采用 VCF 在云端平台建立通讯录，然后再利用云端平台同步到手机本地

针对 IOS 系统我已经完成了一个小程序，导入的数据采用模板「 用户名、手机号码」格式即可，大家下载到本地即可

https://github.com/yidaofeitop/simpleCode/blob/master/js/concatsFormToVCF/concatsFormToVCF.html
