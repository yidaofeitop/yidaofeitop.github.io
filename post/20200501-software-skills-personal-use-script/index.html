自用脚本（全）

2020年05月01日
 read
   Categories: 软件技巧

由于 VPS 在某些时段会出现 IP 地址被封锁的情况，这时候就需要重新配置 SS、Piwigo、宝塔界面等， 为节省相关的时间所以将相关的操作汇总在此！具体内容如下：

  1. ss 环境搭建
  2. LAMP 基础环境
  3. Piwigo 图床搭建
  4. 域名更换

SS 环境搭建

利用「v2-ui」一键搭建 ss 环境，安装脚本如下：

bash <(curl -Ls https://blog.sprov.xyz/v2-ui.sh)

安装完成之后访问链接为：http://xxxx.xxxx.xxxx.xxxx:65432/， 账户名与密码均为admin

Tips：如后续有安装宝塔界面等类似的软件注意开启相关的端口

常用命令：

v2-ui                  # 显示管理菜单 (功能更多)
v2-ui start            # 启动 v2-ui 面板
v2-ui stop             # 停止 v2-ui 面板
v2-ui restart          # 重启 v2-ui 面板
v2-ui status           # 查看 v2-ui 状态
v2-ui enable           # 设置 v2-ui 开机自启
v2-ui disable          # 取消 v2-ui 开机自启
v2-ui log              # 查看 v2-ui 日志
v2-ui update           # 更新 v2-ui 面板
v2-ui install          # 安装 v2-ui 面板
v2-ui uninstall        # 卸载 v2-ui 面板

扩展阅读：

  • github 地址：https://github.com/sprov065/v2-ui
  • 使用说明书：https://blog.sprov.xyz/2019/08/03/v2-ui/

ss 插件

ss 插件大致有 BBR、TLS等，本人仅使用了 BBR 用于加速，安装脚本如下：

wget --no-check-certificate https://github.com/teddysun/across/raw/master/bbr.sh && chmod +x bbr.sh && ./bbr.sh

其后续详细操作访问： 翻墙 chapter 5 - 如何让 SS 更快

LAMP 环境搭建

利用 「宝塔界面」一键搭建 LAMP 环境，安装脚本如下：

yum install -y wget && wget -O install.sh http://123.129.198.197/install/install_6.0.sh && sh install.sh

安装完成之后建议首先完成「面板入口、用户名、密码」修改，否则后续必须使用命令行获取相关 信息，登入进去之后即可一键安装 LAMP

Tips：以上脚本安装环境为 Centos，其他环境安装脚本请访问：https://www.bt.cn/bbs/thread-19376-1-1.html

常用命令

/etc/init.d/bt default   #查看面板入口
service bt stop          #停止
service bt start         #启动
service bt restart       #重启
service bt stop && chkconfig --del bt && rm -f /etc/init.d/bt && rm -rf /www/server/panel  #卸载

Piwigo 环境搭建

Piwigo 为一款比较好的开源图床软件，主要的优势在：图片迁移之后只需要修改域名即可，自建博客神器

安装脚本如下：

wget https://piwigo.org/download/dlcounter.php?code=2.9.4 -O piwigo.zip
ls 
unzip piwigo.zip 
cp -rf piwigo /www/wwwroot/
chmod  777 /www/wwwroot/piwigo

其后续详细操作访问： 个人建站-组件-自建图床 Piwigo
