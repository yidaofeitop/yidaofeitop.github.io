Firefox 浏览器简易教程

2019年01月13日
 read
   Categories: 软件技巧

2019年1月13日 在阅读完弃用 Chrome 改用 Firefox 的几点理由——关于 Chrome 69 隐私丑闻的随想决定将浏览器的主力转移至「Firefox」，其原因最为主要的如下：

  1. Chrome 浏览器与 Google 相关服务结合过于紧密，在后续迁移的过程中更是证明了这一点：查看、控制和删除您的 Google 帐号中的信息
  2. Tor 与 Firefox 强强联合，从而导致 Firefox 在隐私方向的防护力量更强

当然后续也将继续使用 Chrome，毕竟已经习惯他的开发调试工具！

版本问题

首先需要注意以下 Firefox 的版本问题：

  1. 按照国籍来分的话可以分为国内版以及国际版的区别，其主要区别在于
    • 安装包的数字签名不一样
    • 书签等云端数据存储在哪里，两者账号不互通
    • 下载地址不一样
      • 国际版：1. 官网 2. 官方-FTP 服务器
      • 国内版：1. 官网

  2. 按照版本更新频率可以分为：普通版和ESR，其主要差别如下：

    • ESR版本是 Mozilla 专门为那些无法或不愿每隔六周就升级一次的企业打造。Firefox ESR 版的升级周期为 42 周
    • 普通 Firefox 的升级周期为 6 周

    下载地址分别如下：

    • 中国版ESR：http://www.firefox.com.cn/download/#download-esr
    • 国际版ESR：https://www.mozilla.org/en-US/firefox/organizations/all/ ,选择非简体中文即可

大家按照各自的定位下载不同的浏览器即可！

如下载国外普通版本，可能需要调整语言，大家按照使用语言包改变Firefox界面语言教程操作即可！

浏览器基础配置

选项功能

定位「选项」页面，分别配置以下内容（请大家按照自己的需求判断是否要更改）：

  • 「常规-下载」：将下载目录调整到为「公共资料盘」下某个文件夹（关于公共盘请访问Windows 教程 3……最完美的分盘方案）
  • 「常规-Firefox 更新」：调整为「不检查更新……」
  • 「搜索-默认引擎」：调整为「duckduckgo」
  • 「隐私与安全-表单与密码」：全部取消
  • 「隐私与安全-Cookie 和网站数据」：调整为退出浏览器即刻删除「Cookie」并阻止「第三方Cookie」
  • 「隐私与安全-跟踪保护」：调整为始终开启，并将「请勿跟踪」调整为始终开启
  • 「隐私与安全-Firefox 数据收集与使用」：调整为仅发送崩溃报告

about:config 配置

在火狐浏览器地址栏键入about:config回车，打开火狐高级设置，在搜索框中输入下列命令

  1. network.prefetch-next = false

    火狐预加载功能，当载入一个网页后，火狐会预加载你可能会打开的网页链接内容，如果你打开下一个链接，网页就会很快加载，可是很多时候都是多余的，CUP 和宽带浪费率太高。建议双击设置为 false。

  2. browser.sessionhistory.max_total_viewers = 10

    这是页面前进/后退中保存的页面数，默认是-1(无限)，把值修改为10或更小的数

  3. security.dialog_enable_delay = 0

    设置安装扩展时等待的时间(单位毫秒)。改值为0后只要一点击“安装到Firefox”按钮不用等待就可直接安装扩展了。默认值是等待4秒。

  4. config.trim_on_minimize = true （视个人情况使用习惯看是否配置）

    右击空白部分，在「新建」中选「布尔」，输入名称「config.trim_on_minimize」，其值设定为「true」，最小化火狐时释放内存

  5. nglayout.initialpaint.delay = 0

    创建新的整值「nglayout.initialpa0int.delay」，设定为「0」，这个值是延迟显示整个网页的时间，设置为0则即可显示网页，适合网页还未加载完成就先进行阅读的用户习惯

  6. browser.tabs.closeWindowWithLastTab = false

    改为 false——关闭最后一个标签页不关闭浏览器

  7. privacy.resistFingerprinting = true

    此偏好可让 Firefox 更能抵抗指纹辨识

  8. browser.safebrowsing.malware = false

    取消 Google 安全浏览的恶意软件检查，将相关的检查交给安全软件

脚本

油猴 Tampermonkey 的官方网址是：https://tampermonkey.net/

油猴默认是没有安装任何脚本的，你可以通过设置页面的「已安装脚本 - 获取脚本…」来下载各种脚本。

比较常用的下载渠道有三个：

  • Greasy Fork：支持中文，按照今日安装、总安装数、得分、创建日期等的排序方式给出脚本列表，可按脚本生效的网站过滤，每一脚本都有中文介绍，并且详细列出了作者、安装数、更新日期日志、安装使用截图、兼容性、应用到、代码查看等信息描述。
  • OpenUserJS、Userscripts Mirror：这两个网站都没有提供中文界面，Userscripts Mirror 已经停止了更新，用户可以在这个站点找到历史资源。

找到需要的脚本后，会在介绍页面看到安装（install）按钮，点击下载脚本后会自动跳转到安装界面，再点击安装就可以享用脚本了。
