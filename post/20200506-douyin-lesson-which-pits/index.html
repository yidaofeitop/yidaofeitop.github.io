抖音-理论部分-4：抖音有哪些坑

2020年05月06日
 read
   Categories: 互联网运营

上一堂课程主要是了解了「抖音推荐算法」，一般来说后续就是直接讲解怎么实操了，如「抖音养号」……，但是我想要先讲讲抖音有哪些坑，这样的话大家能够尽量少走一些弯路，有时候这将节省你很多的时间与精力！

下文内容主要分为以下几个部分：

  1. 违规违禁规则
  2. 顶层设计方面
  3. 细节执行层面

违规违禁规则

抖音规则主要分为2个部分：《抖音社区自律条约》以及《抖音直播行为规范》

研究以上的一些规则可以避免重大失误，尤其是直播！

追加阅读：平台直播自律白皮书

顶层设计方面

结合自身以及网络上关于「抖音有哪些坑」中的部分内容，梳理出以下的表现形式：

  • 不站在用户的角度发作品，出现自己很嗨皮但是实际的效果却很差
  • 未明确自己的变现方式，前期一味的追求粉丝量，到后面财力难以支持
  • 未对抖音大方向有一定的了解，越做越吃力
  • 未对抖音的变现能力有一定的调查，自我觉得可以，但实际情况是变现之路小的很
  • 未针对自己的特长、资源等做一个全方位的了解，盲目的拍视频、直播等……
  • 频繁修改封面等基础信息
  • 内容创作方面的主题没有任何的垂直度，关联性……

这个方面基本上都可以说是「抖音顶层设计方面」的失误，如要避免这些失误大家可以从以下角度思考：

  • 定位部分：
    • 我是谁？我擅长哪些东西？我有哪些资源？
    • 我内容输出的受众是怎样的一个群体？在这个群体上我能够挣哪些钱……
      这样至少你会对自己要做的内容以及变现方式会有一个比较清晰的了解
  • 对标部分：
    「最好的老师是自己的对手」，基于自己的定位去找相应的对标账号来进一步细化自己的定位以及变现形式等……

细节执行层面

  • 在主页留下电话号码、V信等敏感内容

一般而言建议大家至少在1千粉丝之后再在留电话号码，如想要留的可以通过将抖音昵称改为「类微信号」方式处理

  • 粉丝不过千就实名认证、蓝V认证

实名认证解除需要7天，蓝V认证需自认证之日起一年之后才能解除，所以不要过早的实名认证、蓝V认证等，等待账号权重、运营技巧起来之后再认证岂不是更香！

  • 频繁的更换手机号码、昵称、封面、登录退出等……

在一定程度上会降低账号权重，原因可以去看 抖音-基础理论部分-3：抖音推荐算法-数据加权，所以大家如无必要请不要进行如上操作

  • 采用微信「文件传输助手」作为中介传输成片出现模糊

微信对于视频、图片等均有压缩，大家如果要传输成片建议使用QQ，另外视频清晰也应该做好以下内容： 1. 参数：视频越小越清晰，推荐1080P 60帧（不要用微信传输） 2. 设备：苹果11、XR 3. 滤镜：磨皮可以，滤镜美颜别开惨白色 4. 打光：给侧光或45度光 5. 调节：剪映锐化调节25-30左右

  • 刷粉刷赞

也是很多人讲的一个坑，但是大部分人并没有将原理讲解清楚，我从我的理解说一下：

  1. 抖音流量池分为内部流量池与外部流量池，内部流量池即抖音算法给与的「冷启动」流量，而刷粉则属于「外部流量」，不在算法的考核机制里或者系数极低
  2. 抖音算法的评估参数是「完播率」、「点赞率」、「评论率」、「转发率」，如果某个数值过高的话系统会标记「刷赞」标记，这对于新号来说是一个比较恐怖的事情，但是对于大号来说还好
  3. 如果你有一定的粉丝量之后，抖音是会优先将你的视频推送给「粉丝」的，但是这些粉丝对你的内容并不是真正的感兴趣，所以会导致「完播率」、「点赞率」、「评论率」、「转发率」降低，这样自然会降低视频的自然流量
  • 长视频

很多人觉得发长视频好，15秒做不出东西来，但是没有考虑到完播率，抖音上的算法比较重要的也是完播率，有多人看你的作品，看的怎么样，从完播率上就会去评估你的作品质量和喜好。

什么叫完播率?一个作品的内容是15秒，用户需要用15秒看完了，这就是完播率，如果你的视频很长1分钟，需要花费1分钟的时间才能看完，如果看不完就划走了，那么会判断你的视频不是用户所喜欢的，

所以一般不建议大家去考虑长视频，因为对于整个团队的要求非常之高！大家尽量将视频控制在 15S 之内

类似的不懂「抖音算法规则」的问题还有如：

  • 多拍多发：建议大家保持每2-3天出一个精品即可
  • 搬运视频
  • 采用违禁词等……
  • 视频露出广告、品牌 Logo……
