书籍清单-0-3岁幼儿读物

2019年01月10日
 read
   Categories: 维基

初心

某日婶婶来消息说：「推荐相关0-3 读物」！

初步的想法时这个一个非常艰难的任务，因为自己完全没有这方面的经验，毕竟我还是一只撸代码的单身 🐶

但是想想自己总有这样一天，所以还是硬着头皮上吧！

PS：

  1. 本文全部为网络资料汇总，请大家遵循自己的情况吸收相关的资料！
  2. 本文绝大部分内容为网络摘抄内容，后续为了阅读体验感会保留相关的引用说明，如有侵权请联系本人处理

基础脉络

本部分主要梳理清楚相关的理念问题问题，主要内容为：

  1. 明白为什么买
  2. 如何购买
  3. 选购指南

一起以「用」为主

本部分摘抄自毛爸聊玩具：……（上篇）

  1. 亲子阅读是一件很快乐的事，千万不要被“宝宝5岁前一定要看X本书”、“我一定要给宝宝买2000套绘本”这样的目标绑架。

  2. 有些绘本的确是育儿好帮手，但买绘本不可太功利，绘本带来的亲子时光，比绘本本身的“功能性”要有价值得多。

  3. 想尽一切办法孩子都不爱看绘本，那也没必要焦虑。阅读虽然有好处，但不爱阅读不代表就有坏处。对学龄前的宝宝来说，多去户外玩耍可能比多读绘本更重要。

购买理念

本部分摘抄自毛爸聊玩具：……（上篇）

  1. 打着孩子的旗号给自己买绘本没什么不可以，没人规定绘本只是给孩子看的，成人不能看。好绘本的艺术价值是超越年龄界限的。

  2. 绘本尽量买适龄的，但也不是说就绝对不能超龄。只是一般来说，适龄的绘本孩子在理解方面会更轻松，阅读时获得的乐趣也更多。

  3. 孩子刚开始接触绘本的时候，购买绘本的风格和题材都尽量丰富一些，这样未来孩子对绘本的接受度会更加宽泛，不容易“挑食”。

  4. 对于新接触的绘本种类，可以先买单本的，确定孩子喜欢再整套收入，避免整套吃灰的情况。

  5. 学会利用公共资源。图书馆、绘本馆、书店这些场所多带孩子去逛逛，让孩子亲近书之余，也能更直观地了解孩子的阅读偏好。

  6. 绘本群、闲鱼、同城论坛也可以留意一下，跟妈妈们互通有无，也能让绘本的周转、利用率达到最高。

对味

如果上面的是列书单是「鱼」，那么这部分就是「渔」，具体的要点来自知乎-爱因妈-绘本推荐

  • 有感兴趣的元素
    对于刚开始接触绘本的宝贝，绘本对宝贝来说是一个很陌生的事物。如果在它身上找到一点宝贝认识的或是喜欢的东西，宝贝就会觉得绘本很亲切，愿意去深入了解。

  • 与生活同步：选择绘本和生活同步，宝贝喜欢，妈妈省心
    买绘本不为了看而看，是刚好需要。

    • 玩具玩完了要收拾好！
    • 见到人怎么不打招呼呢？
    • 妈妈叫你，你听到要回答！
    • 不爱刷牙怎么办啊？

  • 举一反三
    宝贝喜欢上一本绘本很多时候是喜欢它的画风或者主人公，买同风格、同系列的绘本来满足宝贝的大胃口吧！
    针对同风格类型可以先买一本试探下小家伙的喜好，如果不喜欢，就别往里砸钱了。

主要以知乎-爱因妈-绘本推荐为主，其他的仅作相关的挑选，其相关的书单请访问以下参考总书目，以下的推荐的书籍如下：

0-6个月

这个时期主要是以视觉刺激为主，听觉、触觉为辅进行感受式阅读。

一般3个月大时，宝宝就具有了三色视觉，但这时候她们最感兴趣还是对比强烈的黑白图案。这个时期的宝贝大多把书当作食物或玩具，所以出现撕书、吃书的现象都是很正常的事。

推荐：黑白卡、彩色卡、布书

推荐书籍：

  • 《小宝宝视觉激发黑白卡①》
  • 《邦臣小红花0岁宝宝视觉激发黑白卡》
  • 贵州人民出版社的《婴儿视觉启智绘本》

另外在全过程可以配合相关的诗和童谣：

  • 果麦文化的《给孩子读诗》
  • 二十一世纪出版集团的《中国童谣》
  • 明天出版社的《牛来了——给0-3岁宝宝的传统儿歌》
  • 神器：点读笔

6-9个月

选书主要以构图简单，色彩鲜艳；文字方面词汇要简单，韵律感要强；内容要有重复性。

这时宝宝辨别物体物象细微差别能力(简称“视敏度”)的发展关键期，此时他们需要的是颜色对比鲜明的图像和玩具。

推荐书籍：

  • 贵州人民出版社的《婴儿视觉启智绘本》
  • 未来出版社的《靓丽精美触摸书系列》

  • 《蹦》、《噗~噗~噗》
    推荐语：知乎-光光

    日本科学绘本大师松冈英达低幼绘本代表作。
    这两本书最大的兴趣点就是照着简单重复的文字发音，这两个字的发音对宝宝特别有吸引力，重复也符合婴儿学习语言的特征。妈妈可以配合动作更换不同的语气音调，鼓励宝宝进行模仿。

9-12个月

选可以选有故事性、趣味性的书籍。像洞洞书、机关书、游戏书不仅可以提高宝贝的阅读兴趣，还可以锻炼宝贝的精细动作，提高专注力。

这个时期宝贝已经是个小人物了，虽然还不会表达，但已经可以理解爸爸妈妈说话的意思了。他们对于书中的图画会看得更仔细，接受到的信息也随之增多。

推荐书籍：

  • 《抱抱》
    推荐语：知乎-晴天pereting
    亲情主题的绘本，适合9个月以上的宝宝。

小猴子在森林里找妈妈，一路看见不同的小动物的妈妈抱宝宝，最后在动物的帮助下自己也找到了妈妈的故事。

  • 《猜猜我是谁系列》
    推荐语： 知乎-唐欢
    这是一套躲猫猫的绘本，通过躲猫猫洞洞吸引宝宝来看，期待藏在下一页的可爱图像。

    这套书的颜色非常鲜艳，每一页都有洞洞，洞洞后面都有图，有奶牛、猫头鹰、斑马等等，特别是每本书 的最后一页，都有意外惊喜（这里我就不剧透了）

    推荐语：知乎-不会飞的彼得潘
    这本书的图案颜色明快、内容丰富，文字简单易懂、韵味十足。它的每一页都有洞洞，洞洞里能看见下一 页的动物和物品的局部，有奶牛、猫头鹰、斑马、火车等，最后一页最特别，是一面小镜子，孩子们能看 见自己的小脸。掀开书里一个个奇妙的洞洞，再加上大人们的生动引导，孩子们一定会被逗得哈哈大笑

  • 《噼里啪啦》系列
    推荐语：知乎-不会飞的彼得潘

    这本书图案可爱生动，文字简洁。书里的每一页都是丰富有趣的大图，还有很多局部折叠的图案。孩子带 着好奇心打开折叠面，会发现里面藏着更有趣的画面，这是多么令孩子愉快的阅读体验啊！每一页都是很 厚的铜版纸，很结实，书角也是圆边的，不会划伤孩子的小手。

  • 《冰冰和波波》
    推荐语：知乎-不会飞的彼得潘

    书中的冰冰是一只可爱的小企鹅，波波是一只憨憨的北极熊。他们俩是一对儿好朋友，共同生活在寒冷的 北极。书中有许多可以滑动的小机关，小朋友随着每一页的翻动，滑动的那些机关，就可以跟随着小企鹅 和小北极熊，一起刷牙、捉迷藏、溜冰……感受生活中每一个熟悉的场景，和冰冰波波一起度过欢乐的时 光。

  • 《点点点》《变变变》
    推荐语：知乎-光光

    这是一本关于颜色认知的有趣的互动游戏书。通过简单的指令，纸上的点按一下会变多、摸一摸会变色、摇一摇还会跑得到处都是……非常有趣。整本书挥洒着魔力，吸引孩子热烈参与，认识了三原色的同时，也丰富了孩子的想象力。《变变变》作为《点点点》后续作品延续了互动游戏的特色，同时增加了对基础颜色混合可产生新的颜色这一知识的认知，在体会阅读乐趣的同时，也增长了美术知识。

1-2岁

1~2岁是宝宝建立自主的阶段。随着自我意识的萌发，宝宝对周遭的事物充满了好奇心，想独立支配环境的愿望也越来越强烈。这时宝贝大多对绘本已经有了自己的喜好，读绘本不仅可以丰富宝贝的知识，更可以激发他们的想象力，一起探索世界吧。

这个时期选择生活习惯、性格培养、情绪管理、认知提升等方面的书都特别好。

推荐书籍：

  • 《小熊宝宝系列》

  • 《猜猜我有多爱你》
    推荐语：知乎-光光

    一大一小两只兔子短短的对话，浓缩生命中最复杂、也是最伟大的情感。小兔子认真地向大兔子表达“我爱你”，并用各种身体动作和看得见的静物表达自己对大兔子的爱。但不管怎么比，永远也比不过大兔子来得多、来得高、来得远。最后，小兔子终于在一片酽酽爱意的包裹下睡去了。

  • 《好饿好饿的毛毛虫》
    推荐语：知乎-不会飞的彼得潘

    书中的毛毛虫爬啊爬，从星期一到星期日吃了一大堆东西，它周一吃了一个苹果，周二吃了两个梨子、周 三吃了三个李子、周四吃了四个草莓……每一个水果上都挖了一个小圆洞，毛毛虫就这样透过书页上的小圆 洞一直吃个不停，吃到最后，变成了一条巨大无比的毛毛虫，然后又变成了一只色彩斑斓的蝴蝶。这是一 本功能性很强的认知书，教给孩子时间、数量的序列关系。而且，它还用鲜亮的色彩表达了化茧成蝶的璀 璨，它是一本让读者感受到希望的书。

    推荐语：知乎-不会飞的彼得潘

    这本书应该很多妈妈都知道，绝对是绘本中的经典，建议每个家长收入。色彩鲜艳画风精美，尤其是书上毛毛虫吃过的小洞洞，对从一岁半左右开始正处于细小事物敏感期的孩子来说简直如获至宝，宝宝总是乐此不彼地把手指伸进每个洞里钻啊钻。有趣的故事设计，把星期、数字、量词、食物、大小、毛毛虫的演变，甚至贪吃多了肚子会痛等这么多的信息量囊括其中，宝宝每一次翻都会有惊喜的发现。

  • 《我爸爸》/《我妈妈》
    推荐语：知乎-光光

    作者用孩子的口吻和眼光描述了一位，喜欢穿着黄格子睡衣的，强壮又温柔，无所不能，带给家庭十足安 全感的爸爸。这会让孩子产生共鸣：我爸爸也像书里说的一样棒，不对，我爸爸比书里的爸爸还要棒！

    《我妈妈》和《我爸爸》非常相似，两本书都是借着孩子天真自豪的口吻，描绘心目中无所不能的爸爸和 妈妈。
    布朗同样运用对比的句子，形容妈妈的各个方面。比如，说妈妈的歌唱像天使，吼叫起来又像狮子；或者 说妈妈像猫咪一样柔软，又像犀牛一样强硬。表达了一个孩子对妈妈深深的崇敬与爱。

2-3岁

这个时期选择具有一定知识性、文学性、艺术性、可以引发宝贝思考的书籍比较好。

这个年龄的宝贝已经具备表达能力，阅读能力更是大幅提升，也有一定的鉴赏能力。可以选择一些百科性的书籍增长见识，也可以选择图文比较美的书籍以艺术熏陶，读一些有深意的内容让宝贝思索。

推荐书籍：

  • 《DK幼儿百科全书》
  • 《可爱的身体》
  • 《是谁嗯嗯在我的头上》
  • 《花婆婆》
  • 《你看起来很好吃》
  • 《不一样的卡梅拉》

  • 《大卫不可以》
    推荐语：知乎-不会飞的彼得潘

    大卫是一个调皮捣蛋的小男孩儿，总是让妈妈和老师不省心。大卫伸着舌头，站在椅子上颤颤巍巍去够糖 罐；大卫一身污泥回家， 客厅的地毯上留下了一串黑脚印；大卫在浴缸里闹翻了天，水流成河；大卫光着 屁股跑到了大街上……在每一页里，大卫都要闯一次祸，旁边都有一句妈妈说的话“大卫，不可以！”

    但是到了最后，大卫擦着眼泪，罚坐在角落里。妈妈对他说：“宝贝来这里。”并给了大卫一个温暖的拥 抱。一个童年恶作剧的故事结束于这样一个爱的动作。让孩子感受到，不管自己多么调皮，当自己伤心 时，妈妈的怀抱永远是他温情的港湾。

  • 知乎-孔阳君-关注绘本心理建设
    关键语句：

    不过同时也有些遗憾的是，大家对于绘本的认识，还停留在绘本有什么用上面，而不是去问如何和儿童一 起读绘本，儿童绘本在儿童心灵治愈中起到什么样的作用。

    ……

    至于绘本有没有用，我还是引用一下《夏山学校》的校长A.S尼尔的一句话：对于儿童的成长来说，最重要 的成长阶段恰好是儿童没有记忆的出生后的四年。

    之所以引用这句话，是因为儿童在成长时期所经历的口欲期、肛欲期等，儿童的原始欲望如果没有被父母 重视、满足的话，会对儿童一生的人格产生消极影响。这就是有些人在青少年后会暴饮暴食发胖、或是咬 指甲（多是因为口欲期没有被满足），而需要接受心理上的治疗。（而尼尔认为，并不是儿童的问题需要 见心理医生，而是他们的父母！）

    所以您问三岁之前给孩子读绘本有没有用，倒更希望您提问，三岁之前应该怎么为孩子读绘本，引导他们 负面情绪发泄出来，埋下一颗温柔的种子。
