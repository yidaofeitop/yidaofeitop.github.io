适配锁体判断基本步骤

问题A.测量侧边条长宽

  1. 210x33->王力锁体
  2. 240x30->问题B
  3. 260x24->问题C
  4. 280x24->问题D
  5. 388x40或388x30->大霸王锁体
  6. 其他规格->请按照“锁体数据列表”判断

问题B.锁舌是否是圆柱

  1. 是->小霸王锁体
  2. 否->国标锁体配侧边条即可

问题C.锁芯距(把手中心到门边距离)为75还是68

  1. 75->盼盼锁体
  2. 68->国标锁体配侧边条即可

问题D.侧边条单孔直角且有4个孔

  1. 是->保德安锁体
  2. 否->国标锁体配侧边条即可

指纹锁安装步骤

  1. 实地判断我方锁体是否与旧锁体是否匹配(如非不匹配则配相应锁体实在没办法可上全自动锁)
  2. 智能锁装锁预备动作：前面板上连接柱、选用适配的方钢
  3. 拆卸旧锁，全部放置在智能锁纸盒中，以便后续整理
  4. 装配新锁体：先上地钩后上天钩，固定螺丝打对角螺丝即可
  5. 装前面板：非全自动前面板上门后上提反锁下压锁体锁回为正常情况
  6. 接线装后面板：注意连接线偏离天地钩，如无位置则挖深或者从锁体中底部牵出线
  7. 处理锁扣板部分：如位置仅是扣板横档不适配，直接使用角磨机切割横档即可，否则换扣板

补充要点

  1. 开孔前尽量使用开孔图或封板确认相关的位置点，选用恰当的开孔钻头开孔
  2. 如为增加的孔，考虑最后如何处理，如无必要不开
  3. 锁体天地钩尽量不要做任何的变形处理，如 处理了请在固定锁体之前确认锁体运转正常
  4. 安装过程中一定要算一下 安装完成后方钢不滑动，或者有滑动但是能够确保前面板与方钢一定有接触
  5. 安装完成后检查 关键配件（方钢+弹簧）是否使用、使用是否恰当（注意是两个步骤），并反复检查整体运转情况
